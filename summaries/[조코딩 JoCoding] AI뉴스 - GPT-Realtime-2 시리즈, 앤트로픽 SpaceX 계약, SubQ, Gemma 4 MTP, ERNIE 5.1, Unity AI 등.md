## **OpenAI GPT Real-time 2 시리즈 공개**

- GPT-5급 추론 능력을 갖춘 음성 모델 3종을 새롭게 공개했음.

- 음성 기반 모델이며 실시간으로 상호작용 가능함.

- 세 가지 모델은 다음과 같음:

- GPT Real-time 2: GPT-5급 추론 능력이 내장된 음성 모델.

- GPT Real-time Translate: 실시간 번역 전용 모델.

- GPT Real-time Whisper: 실시간 음성 전사(음성을 텍스트로 변환) 모델.

- <strong>GPT Real-time 2 상세 내용</strong>

- 기존에는 음성을 텍스트로 변환하고, 그 텍스트를 GPT-5에 넣어 결과를 추출한 후 텍스트로 응답하는 복잡한 절차를 거쳤으나, 이를 하나로 통합했음.

- 응답 속도가 매우 빠름을 시연 영상을 통해 확인할 수 있었음. (예: "안녕" -> "안녕. 반가워. 편하게 말 걸어줘." 응답)

- 도구 호출(Function Calling)도 가능함.

- 플레이그라운드에서 `getStockPrice` 함수를 이용해 "삼성전자 현재 주식 가격 보여 줘."라고 명령하면 실시간으로 삼성전자 종목 코드(005930)를 기준으로 현재 주가(78.65, 원화)를 조회하여 응답했음.

- 자체적으로 함수나 MCP 서버와 연동해 놓으면 실시간 대화 중 펑션 콜링이나 MCP 서버 호출이 가능하다고 함.

## **OpenAI GPT Real-time Translate 및 Real-time Whisper 기능**

- <strong>GPT Real-time Translate</strong>

- 실시간 통번역 전용 AI 모델임.

- 70개 이상의 입력 언어를 13개 출력 언어로 실시간 번역함.

- 번역 품질이 꽤 뛰어나다고 함.

- 외국어 음성이 거의 지연 시간 없이 실시간으로 출력되는 것을 시연 영상을 통해 들을 수 있었음.

- 행사 등에서 귀에 꽂고 실시간으로 듣는 통번역 기기를 API 하나로 구현하여 바로바로 실시간 통번역이 가능해짐.

- <strong>GPT Real-time Whisper</strong>

- 음성이 들어가면 텍스트가 나오는 실시간 전사 모델임.

## **GPT Real-time 시리즈 활용 데모 및 사례**

- <strong>스트리밍 중 실시간 자막 및 그림 자동 생성 도구 (오토프레소)</strong>

- GPT Real-time 2를 이용해 개발된 도구로, 스트리밍 중에 모든 언어로 실시간 자막을 추가하고 설명하는 내용에 따라 그림까지 자동으로 그려주는 기능을 구현했음.

- Real-time Whisper로 실시간 자막이 생성되었음.

- 말한 내용이 슬라이드 형태로 실시간으로 그림으로 표현되어 나왔음.

- 이 도구는 오토프레소(Autopresso)라는 이름으로 오픈 소스로 공개되었음.

- 시연에서 한국어와 영어 음성이 실시간 자막으로 출력되었고, 말하는 내용에 맞춰 슬라이드로 그림이 그려지는 것을 확인할 수 있었음.

- 예시로 조코딩 채널 소개 내용("요즘 메인 콘텐츠로 일주일간 AI 뉴스 요약 전달", "1인 창업자 및 AI 코딩 활용 강의", "유명 인사 인터뷰 팟캐스트 콘텐츠 진행")을 말하자, 해당 내용들이 실시간으로 정리되어 슬라이드로 자동 생성되었음.

- <strong>목소리만으로 자비스 구현 앱</strong>

- 음성 명령으로 앱을 구현하는 '자비스'와 같은 앱을 구현했음.

- GPT Real-time 2와 코덱스 앱 서버를 연동하여 '투두리스트 만들어 줘'와 같은 명령을 내리면 투두리스트가 생성되고, 음성 대화를 통해 코딩으로 구현하는 것이 가능했음.

- <strong>크로맥스(Choin 님 개발)</strong>

- 유튜브 영상의 라이브 실시간 통번역을 제공하는 도구임.

- 실시간으로 영상을 보면서 한국어나 일본어 등으로 번역된 음성을 들을 수 있었음.

## **AI 모델 성능 벤치마크 (Miter) 및 예측**

- Miter는 새로운 모델이 출시될 때마다 인간 전문가들이 몇 시간 동안 할 일을 AI가 처리할 수 있는가를 기준으로 벤치마크 점수를 측정함.

- <strong>과거 벤치마크</strong>

- GPT-5는 전문가의 3시간 23분 작업량을 처리할 수 있었음.

- Opus 4.6은 전문가의 11시간 59분 작업량을 처리했음.

- <strong>최신 벤치마크 (Mitosys)</strong>

- Mitosys는 인간 전문가들이 16시간 동안 계속 일해야 할 작업량을 수행할 수 있다고 측정되었음.

- 놀라운 점은 측정 자체도 어려울 정도로 성능 증가 속도가 매우 빠르다는 것임.

- <strong>소프트웨어 태스크 벤치마크</strong>

- Gemini 3.1이 전문가 3시간 작업량을 처리할 수 있다고 함.

- <strong>미래 예측</strong>

- 이러한 속도라면 GPT-6는 1년 작업, 나중에는 10년 작업도 수행할 수 있을 것으로 예상함.

- 궁극적으로는 인류 문명이 수행했던 모든 작업을 정복할 수도 있다는 유머성 게시물까지 나올 정도로 모델의 결과물이 빠르게 인간을 따라잡고 있다고 함.
