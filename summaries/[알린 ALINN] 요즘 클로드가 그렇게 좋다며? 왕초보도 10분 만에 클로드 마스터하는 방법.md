## **클로드 활용, 왕초보부터 전문가 수준까지**

- 최근 클로드의 높은 평가에 따라 클로드로의 전환을 고려하거나, 다른 AI(ChatGPT, Gemini)와의 큰 차이를 느끼지 못하는 사용자를 위해 클로드 마스터 방법을 정리함.

- 이 영상 하나로 클로드를 처음 접하거나 제대로 사용해 본 적 없는 사용자가 왕초보부터 전문가 수준까지 단계별로 업그레이드할 수 있는 방법을 안내함.

## **클로드가 좋은 이유**

- <strong>뛰어난 성능</strong>

- ChatGPT나 Gemini에 비해 성능이 가장 우수함.

- 동일한 작업을 지시했을 때 훨씬 높은 퀄리티, 정교함, 자연스러운 결과물을 제공함.

- <strong>다양한 기능 통합</strong>

- PPT, 디자인, 코딩, 자동화 등 다재다능하여 거의 모든 작업을 클로드 하나로 처리할 수 있음.

## **클로드 기본 세팅**

- 클로드에 접속 후 왼쪽 하단 프로필을 클릭하여 '설정'으로 진입함.

- <strong>일반 탭 - 클로드 지침 (Custom Instructions)</strong>

- AI가 원하는 방식으로 답변을 주도록 세팅하는 기능임.

- 기존에 사용하던 프롬프트가 있다면 입력하고, 없다면 제공되는 프롬프트(영상 더보기 또는 고정 댓글 링크 참조)를 복사하여 사용 가능함.

- <strong>기능 탭</strong>

- 메모리, 일반, 시각 자료, 코드 실행 및 파일 등의 기능이 있음.

- 어떤 기능인지 잘 모를 경우 일단 전부 켜 두는 것을 권장함.

- <strong>메모리 기능 - 다른 AI 제공 업체에서 메모리 가져오기</strong>

- 다른 AI(ChatGPT, Gemini) 사용자들의 정보를 클로드로 옮겨 오는 기능임.

- '가져오기 시작' 버튼을 누르면 프롬프트가 생성되며, 이를 복사하여 기존 AI에 붙여 넣음.

- 기존 AI가 기억하는 정보들을 정리해 주면, 그 정보를 다시 클로드에 붙여 넣어 기본 세팅을 완료함.

## **클로드 모델 이해 및 활용**

- 클로드에는 세 가지 주요 모델이 있음.

- <strong>Haiku (하이쿠):</strong> 가장 빠른 경량 모델임.

- <strong>Sonnet (소넷):</strong> 기본 모델로, 대부분의 작업을 처리할 수 있음.

- <strong>Opus (오푸스):</strong> 최상급 모델로, 복잡하거나 중요한 작업에 사용함.

- Opus보다 상급 모델인 Fable이 출시되었으나, 성능이 강력하여 미국 정부에 의해 당분간 사용이 금지된 상태임.

- <strong>모델 사용 권장 사항</strong>

- 대부분의 일반적인 작업에는 Sonnet을 주로 사용함.

- 복잡하거나 중요한 작업 시 Opus를 활용함.

- Haiku는 토큰(사용량)이 아깝기 때문에 매우 간단한 작업에는 사용하지 않음. 간단한 작업은 ChatGPT나 Gemini에 맡기는 편임.

- 클로드는 성능이 좋지만 토큰 소모가 빠르므로, 최대한 아껴 쓰는 것이 중요함.

## **클로드 활용 방식 변경하기 (초보자용)**

- 클로드를 검색창처럼 단순히 질문하고 답변 받는 방식으로 사용하면 성능의 10%도 활용하지 못함.

- 클로드가 시킨 일을 잘하는 것이지, 불분명한 지시를 찰떡같이 알아듣는다는 의미가 아님.

- <strong>효과적으로 일을 시키는 두 가지 핵심 습관</strong>

- <strong>1. 클로드에게 먼저 질문하도록 요청하기</strong>

- 프롬프트 마지막에 "대답하기 전에 우선 네가 이 작업을 하기 위해서 필요한 부분을 나한테 질문해 줘"라고 추가함.

- 클로드가 바로 답변하지 않고, 사용자에게 필요한 정보를 질문함 (예: 보고서 대상, 분량, 강조 부분 등).

- 클로드는 단순히 주관식 질문뿐 아니라, 자신이 생각하는 최적의 옵션들을 객관식으로 제시하여 사용자가 선택하기 쉽도록 함.

- 이를 통해 클로드가 사용자의 의도와 맥락을 더 잘 파악하여 퀄리티 높은 답변을 제공함.

- <strong>2. 클로드에게 스스로 답변을 검토하도록 요청하기</strong>

- 클로드가 답변을 제공한 후 "네가 준 답변을 스스로 한번 검토해 줘"라고 요청함.

- 클로드가 자체적으로 결과물을 피드백하고, 필요한 부분을 수정하며, 실수를 잡아내고, 중요한 부분을 강조하여 완성도를 높임.

- 이 두 가지 습관을 통해 클로드를 수동적인 AI에서 능동적으로 업무를 처리하는 AI로 전환시킬 수 있음.

## **나만의 업무 공간 구축 (프로젝트)**

- 왼쪽 탭의 '프로젝트' 기능을 활용함.

- <strong>프로젝트의 정의</strong>

- 나에 대한 정보와 대화 기록을 저장하는 폴더와 같은 공간임.

- 클로드는 대화가 끝나면 이전 기록을 잊으므로, 매번 설명을 반복해야 하는 비효율성을 해소함.

- <strong>프로젝트 생성 및 설정</strong>

- 오른쪽 상단의 '새 프로젝트'를 눌러 '유튜브 대본 작성'과 같이 원하는 역할로 이름을 지정함.

- 각 프로젝트마다 전용 지침(Custom Instructions)과 파일을 입력할 수 있음.

- <strong>프로젝트 전용 프롬프트 생성 방법</strong>

- 클로드에게 "나는 이런 일을 하는데, 이 프로젝트 전용 지침으로 쓸 프롬프트를 만들고 싶어. 나를 인터뷰하듯이 질문해서 내가 하려는 일에 맞는 마스터 프롬프트를 만들어 줘"라고 요청함.

- 클로드가 하는 일, 사용하는 도구, 결과물 형식 등에 대해 질문하면, 사용자가 답변하는 대로 클로드가 설명서와 프롬프트를 생성해 줌.

- 생성된 프롬프트를 복사하여 프로젝트 지침에 입력함.

- <strong>관련 파일 업로드</strong>

- 해당 프로젝트와 관련된 파일들(예: 기존 유튜브 영상 대본)을 전부 업로드함.

- <strong>프로젝트 활용 이점</strong>

- 클로드가 이전 대화 기록과 맥락을 모두 기억하여, 사용자의 말투나 스타일을 분석해 새로운 결과물을 생성함.

- 매번 스타일을 설명하거나 요청할 필요 없이 업무 시간을 대폭 절약할 수 있음.

## **다양한 작업 처리 단계 (텍스트 대화 이상)**

- <strong>1. 다른 서비스와 연동 (커넥터 기능)</strong>

- 왼쪽 탭의 '사용자 지정'에 들어가 '커넥터' 기능을 활용함.

- Gmail, 캘린더, 드라이브, 노션 등 다양한 외부 앱들과 클로드를 연동할 수 있음.

- '플러스 버튼'을 눌러 '커넥터 둘러보기'에서 사용 중인 서비스들을 모두 연동함.

- <strong>연동 이점:</strong> 외부 정보를 가져오거나 클로드 생성 결과물을 외부로 내보내는 것이 가능함 (예: 이메일 내용을 읽고 캘린더에 일정 등록 요청).

- <strong>2. 시각화 기능 활용 (아티팩트 및 클로드 디자인)</strong>

- 클로드의 큰 장점 중 하나로, 다양한 정보를 시각적으로 표현하는 능력이 뛰어남.

- <strong>일반 시각화:</strong> 그래프, 차트, 인포그래픽 등을 생성하여 보고서나 PPT에 활용 시 전문성을 더함.

- <strong>아티팩트 (Artifacts):</strong> 직접 데이터를 상호 작용할 수 있는 대시보드, 웹사이트, 간단한 게임 등을 만들 수 있음.

- <strong>클로드 디자인 (Claude Design):</strong> PPT, 캐러셀, 로고 등을 말로 지시하여 고품질 디자인을 생성함.

- 디자인 감각이 없는 사람도 쉽게 전문적인 결과물을 얻을 수 있음.

- 사용자의 스타일(색깔, 폰트)을 기억하여 한 번 설정하면 매번 설명할 필요 없이 스타일대로 디자인을 생성함.

- 생성된 디자인은 클릭하여 바로 수정(간격, 배치 등)하는 것이 용이함.

- 간단한 프롬프트(예: "이런 이런 주제로 PPT 만들려고 하는데 깔끔하고 세련된 톤으로 만들어 줘")로도 충분히 높은 퀄리티의 디자인을 얻을 수 있음.

## **업무 자동화 단계 (클로드가 스스로 일하게 만들기)**

- <strong>1. 스킬 (Skills) 활용</strong>

- <strong>목적:</strong> 반복적인 작업을 한 번 저장해두고 다음부터는 이름만 불러주면 동일하게 실행되도록 하는 기능임.

- <strong>생성 과정:</strong> 특정 워크플로우(예: 회의록 작성 후 보고서 생성)를 클로드로 작업한 다음, 마지막에 "지금 이 워크플로우를 가지고 다음부터는 내가 회의록만 첨부하면 보고서를 자동으로 생성할 수 있도록 스킬로 만들어 줘"라고 요청함.

- 클로드가 질문을 통해 스킬을 완성해 줌.

- <strong>사용 방법:</strong> 회의록 첨부 후 프롬프트 창에 슬래시(/)를 누르면 생성해 둔 스킬을 불러와 구구절절 설명 없이 한 번에 작업을 완료함.

- <strong>2. 코워크 (Co-work) 활용</strong>

- 컴퓨터에 별도의 클로드 앱을 설치해야 사용 가능함.

- <strong>기능:</strong> 특정 폴더를 클로드에 맡기면, 클로드가 해당 폴더 안의 파일들을 읽고, 정리하고, 새 문서를 만들고, 여러 단계의 작업을 통째로 처리함.

- <strong>예시:</strong> "이 폴더 자료들 정리해서 보고서 PDF로 만들어 줘"라고 요청하면, 클로드가 보고서 기획부터 작성, 디자인까지 완료하여 최종 PDF 파일을 폴더에 생성해 줌.

- <strong>예약 작업 기능:</strong> 매일 하는 일을 예약해두면, 클로드가 해당 시간에 맞춰 작업을 자동 실행함.

- 예시: "매일 아침 7시에 이메일이랑 캘린더를 쭉 훑고 중요 일정을 시간대별로 정리해 줘"라고 예약하면, 아침에 오늘 할 일이 정리되어 있음.

- 코워크는 일을 시키는 것을 넘어 스스로 굴러가는 시스템을 만드는 클로드 활용법의 최종 단계임.

- 코워크 활용법에 대한 자세한 내용은 최근 영상에서 확인할 수 있음.

## **최상위 단계 (운영자 수준)**

- 클로드 코드를 활용하여 나만의 맞춤형 도구나 시스템을 설계하고, 마치 거대한 기업처럼 수많은 에이전트들이 24시간 일하는 운영자 단계임.

- 하지만 이 단계는 일반 직장인의 수준에서는 어렵고 비용도 많이 듦.

- 설명된 업무 자동화 단계까지만 숙달해도 일반인 수준에서 압도적인 생산성과 능률을 확보할 수 있음.

## **클로드의 단점**

- <strong>높은 비용:</strong> 다른 AI들과 요금제 가격은 비슷하지만, 사용할 수 있는 한도(토큰)가 더 적음.

- ChatGPT나 Gemini에 익숙한 사용자들은 한도 초과를 자주 경험할 수 있음.

- 그럼에도 불구하고 많은 사람들이 클로드를 고집하는 이유는 그 가격을 감수할 만큼 성능이 우수하기 때문임.

- 클로드 토큰을 아낄 수 있는 방법이 별도로 정리되어 있으므로 참고하면 좋음.
