## **Remotion Skill과 Cloud Code 소개**

- 지난 15년간 수억 뷰를 기록한 영상들을 제작해왔으며, 현재 영상 제작 분야의 혁신적인 변화를 목격하고 있음.

- Remotion이 최근 프로그래밍 방식으로 영상을 제작할 수 있는 스킬을 출시했음.

- 이 스킬로 모션 그래픽 영상, 새로운 기능 소개 영상 등 무한한 가능성의 영상 제작이 가능함.

- 본 영상의 목표는 Remotion 스킬을 초보자도 쉽게 사용하는 방법을 보여주는 것임.

- Cloud Code에 Remotion 스킬을 설치하는 방법 안내함.

- Cloud Code를 사용하여 Remotion으로 영상을 만드는 방법 시연함.

- Cloud Code에서 스킬을 스태킹하여 영상과 Cloud Code의 기능을 강화하는 방법 설명함.

- Remotion 스킬은 출시 6일 만에 트위터에서 1,300만 이상의 조회수를 기록할 정도로 파급력이 큼.

- 이 스킬은 Cloud Code, Cursor 등 AI 코딩 도구 내에서 프로그래밍 방식으로 영상을 만들 수 있게 함.

- 다른 스킬과 결합하여 영상 품질을 향상시킬 수 있음.

- 예시: 마케팅 스킬이나 자체 제작 영상 제작 스킬과 결합 가능함.

- Cloud Code 커뮤니티 회원들은 최신 풀 리퀘스트(PR) 릴리스를 기반으로 자동으로 영상을 생성하는 방안을 논의 중임.

- PR 승인 및 병합 시, 제품 중심의 설명 영상이 자동으로 생성되어 알림으로 전송되는 시나리오가 있음.

- 무궁무진한 활용 사례가 있음.

- 고객 지원 센터에서 자주 묻는 질문에 대한 해결 방법을 영상으로 자동 제작 가능함.

- 비기술 팀원들도 영상 편집기 없이 간단하게 영상을 제작할 수 있도록 지원함.

- 터미널을 사용하지 않고, 높은 수준에서 모든 과정을 진행할 것임.

## **Cloud Code를 통한 Remotion 스킬 설치**

- 첫 번째 단계로, Remotion 스킬 코드를 복사해야 함.

- Claude Anthropic 데스크톱 앱을 다운로드해야 함.

- 월 20달러 구독료가 있으며, 챗, 코워크, 클로드 코드 기능을 제공함.

- ChatGPT의 경쟁자로 평가받으며, 모바일 앱으로도 이용 가능함.

- 앱에서 로컬 또는 워크스페이스 프로젝트(로컬 컴퓨터 또는 가상 환경)에 접근 가능함.

- Remotion 스킬을 설치할 프로젝트를 선택하거나 새 폴더를 생성해야 함.

- 현재 작업 중인 개인 웹사이트 프로젝트를 선택하여 시연함.

- Claude Code에 "이 스킬 설치를 도와줘"라고 지시함.

- Claude Code가 스킬 설치에 필요한 터미널 명령 실행 권한을 요청함.

- Claude Code가 환경 문제를 자체적으로 수정하고 스킬을 설치함.

- 스킬 설치가 완료되면, 지정된 폴더에 세 개의 에이전트(Cloud Code 등)가 설치되어 스킬 사용이 가능해짐.

- 스킬은 특정 키보드 명령으로 호출할 수 있지만, 더 쉽게 사용할 수 있도록 지원함.

## **Remotion과 Cloud Code로 영상 제작하기 (초기 작업)**

- 이번 주 AI 팟캐스트의 모션 그래픽 스타일 인트로 영상을 제작하는 것을 목표로 함.

- 팟캐스트 이름이 'This Week in No Code'에서 'This Week in AI'로 리브랜딩 중임.

- 영상 제작은 '장면(scene)' 단위로 이루어지며, 각 장면은 레이아웃으로 간주함.

- AI를 활용하여 Claude Code에 입력할 프롬프트를 생성할 것임.

- 현재 사용 중인 인트로 영상의 예시를 보여줌.

- 이전 호스트(David)가 등장하는 기존 스타일의 인트로가 있음.

- 새로운 스타일의 스크린샷을 AI에 제공하여 프롬프트 생성을 도움받을 것임.

- ChatGPT를 활용하여 프롬프트를 생성함.

- "영상 제작에 사용할 프롬프트 생성을 도와줘. 각 프롬프트는 장면별로 나누어져야 해."라고 지시함.

- 추가 컨텍스트 제공: "내 이름은 JJ이고 'This Week in AI' 팟캐스트를 운영해. 주간 팟캐스트로 AI 최신 뉴스, 빌드 방법, CEO 인터뷰 등을 다뤄. 재미있고 애니메이션 스타일의 인트로를 만들고 싶고, 원하는 스타일의 스크린샷을 붙여 넣을게."

- "JJ가 호스트로 등장하여 팟캐스트에서 무엇을 다룰지 보여주는 5~10초 길이의 재미있고 활기찬 인트로를 위한 장면별 아웃라인을 반환해줘."

- 스크린샷을 붙여 넣고 ChatGPT에 프롬프트 생성을 요청함.

- Claude Code에서 Remotion 스킬을 호출하여 첫 번째 장면을 생성함.

- 생성된 프롬프트를 사용하여 "이 스킬을 사용하여 Remotion에서 이 장면을 만들어줘."라고 지시함.

- 이것은 클라우드 코드에서 스킬이 일반적으로 작동하는 방식의 예시임.

- 스킬은 로컬 환경에서 설정되며, 다양한 스킬을 추가하고 직접 빌드할 수 있음.

- 스킬 호출 시점을 정하고, Cloud Code가 특정 지침(마케팅, 영업, 카피라이팅, 영상 편집 등)에 따라 해당 스킬을 사용하도록 할 수 있음.

- Cloud Code는 사용자 대신 로컬 머신의 터미널에서 명령을 실행할 권한을 요청함 (데스크톱 앱으로 고수준 접근성 유지).

- Cloud Code는 프로젝트를 분석하고, Remotion 프로젝트를 생성할 위치(서브폴더 선택)를 물어봄.

- Cloud Code가 사용자를 대신하여 머신에서 서브폴더를 만들고, 영상 제작을 위한 기초를 다짐.

- 종속성을 설치하고, 장면 구성 요소와 루트 파일을 생성하기 시작함.

- 첫 번째 장면이 완성되면 나머지 장면들도 지능적으로 처리하도록 지시함.

- 진행 상황: 모든 빌드 및 실행이 완료됨.

## **영상 검토 및 개선 (자산 추가)**

- <strong>초기 미리 보기 및 오류 수정:</strong>

- 로컬 호스트를 열어 프로젝트를 로컬에서 실행하고 미리 보기 창을 띄움.

- 초기 미리 보기에서 오류가 발생했음을 확인하고, 해당 오류 메시지를 Cloud Code에 붙여 넣어 수정을 요청함.

- Cloud Code가 오류를 검토하고 수정 작업을 진행하여 문제를 해결함.

- 콜드 오픈, 호스트 등장, 쇼 커버, 타이틀 카드, 전체 인트로 등 다양한 장면들이 생성되었음을 확인함.

- 전체 인트로를 미리 보면 "로딩 대기" 메시지와 함께 빠르게 재생되지만, 아직 완성도가 높지 않음을 인지함.

- Remotion 편집기에서 시퀀스를 확인하고 편집할 수 있음.

- <strong>호스트 이미지 추가:</strong>

- 현재 영상은 JJ의 이미지를 가져오려 하지만, 자산이 없어 플레이스홀더만 표시됨.

- 리포지토리에 "public"이라는 새 폴더를 생성해야 함.

- 호스트(JJ)의 이미지를 찾아서 Remotion 영상 프로젝트 내의 "public" 폴더로 드래그하여 이동함.

- Cloud Code에 "JJ용 이미지를 public 폴더에 추가했으니, 플레이스홀더 대신 그 이미지를 사용해줘."라고 지시함.

- Cloud Code가 해당 자산을 업데이트하여 JJ의 실제 이미지가 영상에 나타나도록 함.

- <strong>배경 영상 자산 생성:</strong>

- 현재 영상은 배경에 움직임이나 질감이 부족함.

- ChatGPT로 돌아가 "각 장면에 더 많은 움직임을 추가할 수 있는 보완적인 배경 스톡 영상(3~5초 길이의 AI 프롬프트)을 생성하는 것을 도와줘."라고 요청함.

- Replicate 플랫폼(최신 모델을 사용해 AI 영상을 생성하는 곳, 프롬프트당 지불 방식, 예: Google V3)을 활용함.

- ChatGPT가 생성한 프롬프트를 Replicate에 입력하고, 영상 길이를 4초로 설정함.

- 여러 배경 영상을 동시에 생성함.

- 생성된 영상들을 다운로드함 (초록색: 성공, 파란색: 진행 중, 빨간색: 실패).

- 다운로드한 영상들을 프로젝트의 "public" 폴더로 드래그하여 이동함.

- 영상 파일 이름을 "intro", "middle", "ending"으로 변경하여 각 장면에 매칭시킴.

- Cloud Code에 "각 장면에 내가 생성한 배경 영상을 사용해줘. 인트로: public/intro, 중간 장면: public/middle, 마지막 장면: public/ending."이라고 지시함.

- Cloud Code가 요청을 처리하여 영상에 배경 움직임을 추가함.

- 결과적으로 영상에 배경 움직임이 추가되었지만, 여전히 개선의 여지가 있음.

## **스킬 스태킹: Wave Speed AI로 영상 제작 향상**

- 현재 방식은 Replicate로 수동으로 자산을 생성하고 폴더에 드래그하는 방식임.

- 자산 생성을 직접 통합하는 것이 목표임.

- Wave Speed AI를 소개함: 여러 서비스(Eleven Labs 음성, Nano Banano 이미지, Google V3 영상)를 하나로 통합하는 API.

- <strong>Wave Speed AI 스킬 생성:</strong>

- Cloud Code에 "Wave Speed AI용 새 스킬을 만들어줘. API 키를 줄게. 이 스킬은 이미지 생성에 Nano Banano, 영상 생성에 Google V3, 보이스오버 생성에 Eleven Labs를 사용할 거야."라고 지시함.

- "Remotion과 연동하여 작동하는 스킬을 만들어줘. Remotion이 영상 자산을 생성할 때마다 이 스킬을 사용해 필요한 영상 이미지나 보이스오버 자산을 생성할 수 있도록 해줘."

- (화면 밖에서) API 키를 입력함.

- Cloud Code가 Wave Speed API를 활용하여 Nano Banano, V3, Eleven Labs를 통합한 스킬을 생성함.

- <strong>새 스킬을 활용한 보이스오버 추가:</strong>

- 인트로에 보이스오버 오버뷰를 추가하는 것이 목표임.

- Cloud Code에 "방금 생성한 Wave Speed 스킬을 사용해서, Eleven Labs로 팟캐스트 청중에게 인트로를 소개하는 스크립트를 만들어줘. 30대 남성인 나와 비슷한 목소리를 사용해서 전체 보이스오버를 만들고 Remotion 영상에 추가해줘."라고 지시함.

- <strong>스킬 스태킹 시연:</strong> Remotion은 영상을 만들고, Wave Speed 스킬은 Eleven Labs를 사용하여 보이스오버를 생성함.

- Cloud Code가 제시한 계획에는 "안녕하세요, JJ입니다. AI 뉴스, 빌드, 창업자 이야기. 이번 주 AI. 시작해볼까요."와 같은 스크립트가 포함됨.

- 생성된 스킬을 사용하여 보이스오버 오디오를 생성하고, Cloud Code가 오디오를 다운로드하여 자산 폴더에 추가함.

- 보이스오버가 영상에 추가됨.

- 보이스오버가 포함된 최종 영상 미리 보기.

- <strong>결론 및 향후 가능성:</strong>

- 실제로 촬영하거나 수동으로 편집하지 않고, AI 자산, AI 보이스오버, 스택된 스킬만을 사용하여 영상이 제작됨.

- 무한한 프로그래밍 방식의 활용 사례가 있음: 인터넷에서 아이디어를 수집하고, Cloud Code가 반복적으로 영상을 생성하며, 자동으로 게시하는 스킬을 추가할 수 있음.

- 터미널이나 코딩 없이, 단순히 Cloud Code와 대화하는 방식으로 모든 작업이 가능함.

- 권장 사항: Cloud Code 구독($20/월)을 통해 Remotion 스킬을 설치하고, 자신만의 스킬을 만들어 라이브러리를 구축할 것.

- 구독 및 Cloud Code 개발자 커뮤니티 지원을 독려함.
