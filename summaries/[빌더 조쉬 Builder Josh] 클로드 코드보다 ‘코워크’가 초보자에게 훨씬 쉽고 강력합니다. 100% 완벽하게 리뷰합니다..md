## **클로드 코워크 소개**

- 클로드 코워크는 일반인들을 위한 클로드 코드 버전이라고 보면 됨.

- 윈도우나 맥용 프로그램을 컴퓨터에 다운로드 받아 직접 클로드 코드의 거의 모든 기능을 쉬운 인터페이스로 활용할 수 있는 도구임.

- 클로드 코드는 설치가 어렵고 서브 에이전트나 스킬 개념 이해가 어려우나, 클로드 코워크는 비개발자나 일반인에게 훨씬 적합하다고 판단됨.

- 설치가 매우 쉬워 클로드의 강력함을 100% 활용할 수 있음.

- 클로드 코워크는 PC 내에서 AI가 모든 업무를 수행하는 도구와 같음.

- 파일이나 데스크탑에 있는 파일들을 분석하고 만들고 자동화할 수 있음.

- 브라우저 컨트롤이 가능하여, 코워크 명령만으로 AI가 직접 브라우저를 띄워 클릭하며 업무를 자동화할 수 있음.

- 외부 앱을 '커넥터'로 연동할 수 있음.

- 예시: Gmail, 캘린더, 노션 등과 연결하여 메일 분석, 자동 답장, 생산물 저장, 팀 공유 등을 할 수 있음.

- 클로드 코워크 사용자들은 30% 정도의 운영 비용 절감과 업무 처리 속도 및 데이터 연동 향상 효과를 얻었다고 함.

- <strong>클로드 코워크 vs 클로드 코드 비교:</strong>

- <strong>클로드 코워크:</strong> 비즈니스 일상 업무 실행에 유용하며, 직장인이나 일반인이 편리한 자동화를 실행할 때 적합함. 브라우저나 파일 앱 자동화와 같은 간단한 일상 업무에 적합함.

- <strong>클로드 코드:</strong> 커스텀 소프트웨어 구축이나 고도화된 자동화를 구현할 때 적합함. 터미널 기반에서 작동하며 AI에게 100% 권한을 부여할 수 있어 개발자나 전문 직업군이 영상 제작, 소프트웨어 개발, 마케팅 자동화 등 고도화된 기능에 활용하기 좋음.

- 개인적으로 클로드 코드를 먼저 시작하기보다는 클로드 코워크로 클로드의 강력함을 체험해 보고, 어려운 작업이 필요할 때 클로드 코드로 넘어가 보는 것을 추천함.

## **다섯 가지 핵심 활용 사례 개요**

- <strong>1. 파일 관리:</strong> 데스크탑 파일 정리, 인터넷 자료 분석 및 수집, 파일 생성 및 분석.

- <strong>2. 브라우저 컨트롤:</strong> 리서치, 문서 생성.

- <strong>3. 외부 앱 연동 (커넥터):</strong> 노션, Gmail, 슬랙 등을 연동하여 일상 자동화.

- <strong>4. 스킬:</strong> 반복 업무를 스킬로 저장하여 자동 실행.

- <strong>5. 플러그인:</strong> 외부 앱 연동과 커스텀 스킬들을 조합하여 한꺼번에 자동화하는 번들링 도구. 여러 업무를 묶음화하여 한 번에 처리할 수 있는 종합 자동화 패키지임.

## **클로드 코워크 설치 및 초기 설정**

- 클로드 데스크탑 앱을 검색하여 "Download Claude" 주소에서 컴퓨터 OS에 해당하는 파일을 다운로드 받음.

- 윈도우에서도 최근 코워크가 출시되었음.

- <strong>주의사항:</strong> 윈도우에서 클로드 코드를 사용하던 경우, 데스크탑 앱 설치 시 클로드 코드보다 데스크탑 앱이 먼저 명령어로 불러와지는 오류가 발생할 수 있으므로 별도의 조치를 취해야 함.

- 다운로드 후 설치를 완료하면 클로드 데스크탑 앱 화면이 나타나며, 여기서 '코워크' 섹션으로 이동함.

- 코워크 화면에서 AI가 컴퓨터의 모든 내용을 읽고 파일을 생성하며 작업을 수행할 수 있음.

## **1단계: 파일 관리 (단일 파일 컨트롤)**

- 컴퓨터 폴더 내 파일들을 제어하는 것부터 시작함.

- <strong>예시 1: 이미지 파일 정리</strong>

- '다른 폴더 선택'을 눌러 정리하고 싶은 폴더를 지정함. (예: 다운로드 폴더)

- 명령어: "이 폴더에 있는 이미지 파일들을 이미지 폴더로 만들어서 다 넣어 줘."

- 결과: 즉시 '이미지 폴더'가 생성되고 해당 폴더 내 이미지 파일 187개가 모두 그 안으로 옮겨짐.

- <strong>예시 2: 파일 읽기 및 요약</strong>

- '다른 폴더 선택'을 눌러 요약하고 싶은 폴더를 지정함. (예: 빌더 조시 유튜브 폴더)

- 폴더 변경 허용을 누름.

- `Shift + S`를 눌러 파일 목록을 확인하고, 특정 폴더(예: 기부 강의 자료)에 대한 요약을 요청함.

- 명령어: "이 폴더에 있는 자료들을 요약해서 알려 줘."

- 결과: 폴더 내 파일들(입력서, RFP PDF, SaaS 서비스, 시연 자료 등)의 내용과 상위 폴더의 관련 파일들까지 정확하게 요약하여 알려줌.

- 이는 회의록이나 문서 파일들이 많을 때 한꺼번에 요약하여 쉽게 파악하는 데 유용함.

## **2단계: 멀티파일 생성 및 지시**

- <strong>예시 1: 영수증 분석 및 엑셀 파일 생성</strong>

- '새 작업'을 눌러 새로운 작업을 시작함.

- 영수증 파일들이 있는 폴더를 선택함. (예: 10개 정도의 영수증 파일이 있는 폴더)

- 명령어: "이 폴더 안에 있는 영수증들을 분석하고 엑셀 파일로 만들어 줘."

- 결과: 영수증들을 모두 분석하여 엑셀 파일로 정리해 줌. 파일명, 매장명, 주소, 주문 항목, 비용, 합산 비용까지 자동으로 추출 및 정리해 줌. 수기로 영수증을 처리할 필요가 없어 생산성이 매우 높아짐.

- <strong>예시 2: 특정 웹사이트 조사 및 멀티 형식 보고서 제작</strong>

- 명령어: "빌더 조쉬에 대해서 조사해 줘. 메일리 뉴스레터 주소도 있고, 빌더 조쉬라는 사람이 어떤 사람인지 조사하고 나에게 PDF 그리고 DOCX 그리고 PPTX 파일로 보고서를 제작해 줘."

- 진행 과정: 웹 리서치 수행, 페이지 조사 후 PDF, DOCX, PPTX 파일을 순차적으로 (또는 병렬로 지시 가능) 생성함.

- 결과: 빌더 조쉬에 대한 보고서(DOCX, PPTX)를 컴퓨터 폴더에 다운로드 받아 볼 수 있도록 함.

- 인물 소개, 주요 활동 영역, 소셜 미디어 주소 등을 잘 조사하여 보여줌.

- PPT 디자인은 수동으로 손봐야 할 필요가 있을 수 있으나, 보고서 파일 자체는 생성됨.

- 이는 특정 인물, 웹사이트, 경쟁사 리서치 등을 자동화하여 컴퓨터에서 바로 보고서 형태로 받아보는 데 유용함.

## **3단계: 커넥터 활용 및 스케줄링**

- 외부 도구를 연동하여 생산성을 높이는 방법임.

- <strong>커넥터 연결 방법:</strong>

- 코워크 화면에서 '+' 버튼을 누르고 '커넥터'를 선택함.

- '커넥터 관리'로 들어가 '커넥터 둘러보기' 탭에서 원하는 외부 도구(슬랙, 피그마, 캠바, Gmail, 캘린더, 노션 등)를 선택하여 연동함.

- <strong>예시 1: Gmail 연동 및 활용</strong>

- Gmail을 연동해 놓으면 읽기/쓰기가 모두 가능함.

- 명령어: "내 Gmail에서 새로운 메일 도착했는지 알려 줘."

- 결과: 읽지 않은 새 메일의 개수와 주요 내용(유튜브 광고 현황, 보안 알림, 피그마 구독 결제 등)을 알려줌.

- 명령어: "모든 이메일의 대략적인 내용을 요약해서 알려 줘."

- 결과: 대략적인 메일 내용들을 요약하여 보여줌.

- 이메일 관련하여 읽기, 검색, 요약, 분석, 중요한 메일 필터링 등 다양한 작업을 할 수 있음.

- <strong>예시 2: 슬랙 채널 대화 분석 및 공유</strong>

- 슬랙 커넥터를 연결함.

- 명령어: "슬랙 커넥터로 슬랙 내에 있는 제너럴 채널에 최근 일주일 간의 모든 대화록을 분석하고 PDF로 저장한 뒤 그걸 다시 슬랙으로 보내 줘."

- 결과: 채널 내 모든 대화록을 분석하여 요약하고, '클로드'가 슬랙에서 메시지를 보내는 것을 허용하면 '채널 대화 분석 리포트'를 슬랙 채널(또는 DM)로 전송해 줌.

- 리포트에는 총 메시지 수, 참여 멤버, 핵심 인사이트, 주요 공유 콘텐츠 등이 포함되며, 'Sent using Claude' 워터마크가 붙음.

- 향후 클로드 코드(또는 코워크)를 통해 주 1회 자동으로 리포트가 발간되도록 자동화 설정이 가능할 것으로 보임.

- <strong>예시 3: 슬랙 분석 리포트를 노션으로 송부</strong>

- 노션 커넥터를 연결함.

- 명령어: "방금 보낸 요약을 노션에도 송부해 줘."

- 결과: 슬랙 채널 대화 분석 리포트가 노션 페이지에 저장됨. 회의록이나 보고서 등을 노션에 자동 저장하는 데 유용함.

- <strong>기타 커넥터 활용:</strong>

- 캘린더: 클로드 코워크 내에서 일정 관리 중 캘린더에 바로 등록 가능.

- 구글 드라이브: 파일들을 불러와 클로드 코드 내에서 작업한 뒤 구글 드라이브에 바로 업로드 가능.

- <strong>새로운 기능: 예정됨 (Scheduled tasks)</strong>

- 유료 사용자에게 제공되는 기능으로, 매일 정해진 시간에 원하는 작업을 자동 실행할 수 있음. (예: 보고서, 경쟁사 동향, 프레젠테이션 등을 슬랙으로 전달)

- <strong>예시: 매일 아침 AI 뉴스레터를 슬랙으로 받아보기</strong>

- '새 작업'을 누르고 '데일리 AI'를 선택, 설명에 "오늘의 신규 AI 소식 받아보기"라고 입력.

- 프롬프트: "매일 새로운 AI 소식을 요약해서 알려 줘."

- 모델: Sonnet (비교적 쉬운 작업에 적합).

- 빈도: 매일 아침 9시로 설정하고 저장.

- 생성된 '데일리 AI' 작업을 '지금 실행'하여 테스트하면, 당일의 최신 AI 뉴스(엔비디아, 구글 재미나이 등)를 요약하여 보여줌.

- 이를 확장하여 커넥터와 연동 가능. 명령어: "내일 아침 9시에 새로운 AI 소식을 뉴스레터로 만들어서 나에게 슬랙으로 보내 줘."

- 결과: 뉴스레터를 어떤 슬랙 채널로 보낼지 물어보면 다이렉트 메시지(DM)를 선택. 매일 아침 9시에 AI 뉴스레터를 만들어 슬랙 DM으로 보내는 스케줄이 생성됨.

- '지금 실행'하여 테스트하면, 당일의 AI 뉴스레터(주요 7개 뉴스)를 슬랙 메시지로 깔끔하게 전달해 줌. 백그라운드에서 자동 실행되므로 사용자가 매일 확인할 필요 없음.

## **4단계: 스킬 사용**

- 스킬은 특정 작업을 반복적으로 사용할 때 쓰는 도구로, 반복 작업을 모듈화하여 클로드에게 추가하는 지식 패키지임.

- AI가 매번 다른 결과를 줄 수 있는 문제를 해결하여 일관된 품질의 결과물을 보장함.

- 스킬은 `skills.md` 파일 형태로 주로 생성됨.

- 컨텍스트 전달 시, 예를 들어 PPT 스킬에 디자인 파일을 별도로 넣어두면 해당 디자인으로 PPT를 제작할 수 있음.

- 공식 스킬 소개 영상 시청을 추천함. (예: 폴더에 스킬, 가이드라인, 내용이 들어가 PPT 장표를 만드는 과정)

- <strong>예시: URL을 통해 뉴스레터, 링크드인, 쓰레드 글을 한꺼번에 제작하는 스킬 생성</strong>

- 명령어: "특정 URL을 넣으면 뉴스레터와 링크드인 쓰레드 글을 한꺼번에 제작해 주는 스킬을 만들어 줘." (유튜브 URL 포함 가능)

- 폴더 지정 후 시작.

- 클로드가 스킬의 출력물에 대해 질문함. (예: 콘텐츠 생성 언어: 한국어, 최종 결과물 형태: 마크다운 파일, 뉴스레터 스타일: 전문적이고 인사이트 중심)

- 결과: 'URL to Content'라는 이름의 스킬이 완성됨. 스킬 폴더가 제작되고 아웃풋 테스트가 가능함.

- 테스트 결과: 원본 URL을 기반으로 뉴스레터 콘텐츠, 링크드인 포스트(태그 포함), 쓰레드 글을 간단하게 생성해 줌. 결과물은 튜닝이 필요할 수 있으나 명령을 잘 따름.

- 스킬 기능 요약: URL 하나를 입력하면 뉴스레터, 링크드인, 쓰레드 글을 한국어로 동시에 생성하며, 유튜브 URL의 경우 자막 추출 후 변환까지 해줌.

## **5단계: 플러그인 만들기**

- 플러그인은 스킬, 커넥터, 그리고 충분한 에이전트 등을 한데 묶은 '자동화의 집합'으로, 복합적인 자동화에 사용됨.

- 커넥터와 스킬을 함께 사용하고 싶을 때 유용함.

- <strong>플러그인 관리 방법:</strong>

- '+' 버튼을 누르고 '플러그인'을 선택, '플러그인 관리'로 들어감.

- 마케팅 플러그인처럼 특정 플러그인 내에서 어떤 커넥터와 스킬을 사용하는지 묶음으로 확인할 수 있음.

- <strong>플러그인 탐색 및 추가:</strong>

- '플러그인 추가'를 누르고 '플러그인 탐색'으로 들어가면 엔스로픽에서 제공하는 공식 플러그인들을 볼 수 있음.

- CS, 데이터 분석, 금융, 법무, 마케팅 등 여러 작업이 복합적으로 필요한 분야에서 활용 가능함.

- <strong>예시: 콘텐츠 제작 및 배포 플러그인 생성</strong>

- 명령어: "플러그인을 만들고 싶어. URL을 하나 넣으면 그걸 가지고 슬랙과 노션의 자동으로 저장되고 뉴스레터, 블로그, 링크드인, 쓰레드 등으로 한꺼번에 콘텐츠가 제작되는 형태로 제공하고 싶어."

- 클로드가 콘텐츠 추출 방식(자막 기반), 생성할 콘텐츠 종류(모두 선택), 플러그인 사용 목적(개인 사용) 등을 질문함.

- 결과: 'Re-purpose' 및 'Distribute'라는 커맨드를 포함하는 플러그인이 생성됨.

- 'Re-purpose' 커맨드에 URL을 붙여 넣어 콘텐츠 제작을 시작함.

- 제작된 콘텐츠(요약, 블로그 포스트, 링크드인 포스트, 쓰레드/X 포스트, 뉴스레터)를 확인하고 수정할 부분이 있으면 피드백함.

- 'Distribute' 커맨드로 콘텐츠를 배포함. 명령어: "생성된 콘텐츠를 슬랙으로 보내 주세요."

- 슬랙 채널(DM)을 선택하면, 블로그 포스트부터 링크드인 포스트, 쓰레드, 뉴스레터까지 모든 콘텐츠가 슬랙 DM으로 전송됨.

- 이는 개인 채널이나 팀 채널에 모든 콘텐츠를 즉시 전송할 수 있으며, 피그마와 함께 사용하면 피그마 슬라이드, PPT, 구글 프레젠테이션 등도 제작할 수 있어 광범위한 자동화가 가능함.
