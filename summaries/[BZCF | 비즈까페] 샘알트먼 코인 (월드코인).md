## **현재 진행 상황 및 '인간 증명' 문제**

- 매 2초마다 새로운 사용자가 네트워크에 가입하고 있음.

- 총 사용자 1,500만 명을 달성했으며, 이 중 700만 명이 Orb로 본인 인증을 완료함.

- 네트워크 밀도가 중요하며, 이는 개발자들에게 유용한 제품을 구축하는 데 필수적임.

- 산티아고와 리스본에서는 7명 중 1명, 아르헨티나 부에노스아이레스에서는 성인 3명 중 1명이 인증을 완료했음.

- 최종 목표는 서비스를 모든 인류에게 제공하는 것임.

- 튜링 테스트는 지난 12개월 동안 통과되었음.

- 과거 수십 년간 AI가 언젠가 통과할 중요한 이정표로 여겨졌지만, 이제 현실이 됨.

- 실시간 생성 비디오 콘텐츠 및 사진의 시대가 도래하고 있음.

- 대부분의 경우 매우 흥미롭고 콘텐츠를 개선할 것이지만, 사회 운영 방식에 대한 재고가 필요함 (선거, 민주주의, 소셜 네트워크).

- 새로운 시험인 '인간 증명(Proof of Human)'이 지금 필요하다고 생각함.

- '인간 증명' 문제는 처음 생각보다 훨씬 어려움.

- 휴대폰 카메라, 정부 ID, Face ID 등 즉각적인 해결책은 여러 이유로 실패함.

- 문제 해결을 위해 물리적 하드웨어(Orb)를 전 세계에 배포해야 한다는 매우 어려운 결론에 도달함.

- 이 하드웨어는 개방형이 되어야 하며, 많은 다른 회사나 단체가 기기 제작에 참여할 수 있어야 함.

- 기존 하드웨어 기기에 통합될 수도 있음.

- 현재 인프라와 시스템으로는 이 문제를 해결할 수 없음.

- 'Orb'를 개발했음.

- 전 세계 규모에서 '인간 증명' 문제를 해결하는 포괄적이고 안전하며 익명적인 방법임.

## **차세대 Orb 및 운영 확장 전략**

- 새로운 Orb를 발표함.

- Orb는 사용자가 프로젝트를 처음 접하고, 본인이 인간임을 증명하는 매우 특별한 경험의 중요한 부분임.

- Orb가 배치될 공간('Orb spaces')은 사원과 같은 곳이자 공공 공간의 가치를 반영함.

- 환영하고, 단순하며, 평화롭고, 사용자, Orb, 그리고 인증 경험에 집중함.

- 새로운 Orb의 특징:

- 아름답게 디자인되었으며, 처음부터 '규모'를 염두에 두고 설계됨.

- 소프트웨어 스택을 완전히 재작성하여 인증 속도를 3배 빠르게 만듦.

- 부품 수를 30% 줄여 조립 속도를 2배 높임.

- 초기에 Orb 생산 능력을 3배 늘릴 예정이며, 새로운 조립 파트너 및 전 세계 지점 확대로 추가 증대 계획이 있음.

- 탈중앙화된 Orb 제조라는 최종 목표를 향해 나아가고 있음.

- 엔비디아 젯슨(NVIDIA Jetson) 아키텍처를 채택하여 이전 세대 대비 AI 성능이 5배 향상됨.

- 초당 100조(Trillion) 연산이 가능하며, 이는 정교한 AI 모델을 기기 내에서 완전히 실행하고 더 빠르게 인간성을 증명할 수 있게 함.

- 투명성이 강화됨: 운영 지침이 담긴 SD 카드를 완전히 분리할 수 있어, 누구나 SD 카드 내 지침과 공개된 소스 코드를 비교하여 감사할 수 있음.

- 더 많은 Orb의 필요성: 현재보다 약 1,000배 더 많은 Orb가 필요함.

- Orb 접근성 확장을 위한 전략:

- <b>운영 국가 확장</b>: 더 많은 국가로 운영을 확대함.

- 부에노스아이레스와 멕시코시티에 플래그십 지점을 오픈했음.

- 이곳은 프리미엄 인증 경험을 제공하며, 질문에 답할 직원이 상주함.

- <b>셀프 서비스 인증</b>: 기존 소매점에 Orb를 설치함.

- 동네 편의점이나 카페와 같은 장소에 Orb를 배치함.

- 사람들이 소매 환경에서 Orb를 보며 프로젝트를 발견하거나, 인증을 위해 특정 소매점을 방문할 수 있게 함.

- 이는 Orb 위치를 확장하고 지역에 더 많이 분산시키는 데 핵심적임.

- <b>Orb 온 디맨드(Orb on Demand)</b>: 버튼 하나로 Orb를 요청하여 배달받을 수 있는 새로운 기능임.

- 피자 배달과 유사하며, WPPY(라틴 아메리카 1위 배달 서비스)와의 파트너십을 통해 기존 배달 차량으로 Orb를 원하는 누구에게나 전달함.

- <b>커뮤니티 운영자(Community Operators)</b>: 누구나 새로운 Orb를 빌리거나 구매하여 커뮤니티 구성원을 인증할 수 있음.

- 2025년 봄부터 사전 주문한 Orb가 배송될 예정임.

- 이날 행사에 참여한 모든 사람에게 Orb를 제공하여 첫 번째 커뮤니티 운영자가 되도록 함.

- World ID를 사용하여 공정하게 Orb를 배포하고 중복 수령을 방지함.

## **World ID 3.0: 익명 인터넷 인간 레이어**

- World ID는 인터넷을 위한 익명 인간 레이어가 될 것임.

- 단순한 계획이 아니며, '프로토콜'이라는 용어 사용은 모든 인류에게 혜택을 주는 시스템을 구축하겠다는 약속을 의미함.

- 개방적이고, 무허가적이며, 분산화되어 있고, 모든 개발자, 운영자, 제조업체가 기존 사업을 강화하거나 새로운 사업을 구축할 수 있도록 접근성을 제공함.

- 익명 인간 레이어 구축은 1년 전만 해도 불가능해 보였음 (700만 명, 목표 10억 명 규모).

- <b>AMPC(Anonymized Multi-Party Computation)</b> 기술을 통해 이를 가능하게 함.

- 여러 주체가 데이터를 분산 보유하며, 그 데이터가 고유한지 여부만 알 수 있고, 개인 데이터에 대한 다른 정보는 알 수 없게 함.

- 데이터는 조각으로 나뉘어 여러 당사자와 공유되며, 어떤 단일 주체도 데이터를 재구성할 수 없음.

- 중앙 집중화되지 않은 신뢰할 수 있고, 전 세계적으로 분산된 독립적인 주체들이 데이터를 보유함.

- 현재 3개의 파트너(캘리포니아 기반 Berkeley Center for Responsible Decentralized Intelligence, 독일 기반 FAU, 글로벌 블록체인/암호화폐 보안 기업 Nethermind)가 인프라 구축 및 데이터 온보딩을 시작함.

- 취리히 대학과의 파트너십도 최종 조율 중임.

- 올해 11월까지 모든 World ID 데이터는 이들 파트너에게 분산 보관되어, 누구도 데이터의 고유성 여부 외에는 알 수 없게 될 것으로 예상함.

- AMPC의 고유성과 익명성을 활용한 새로운 사용 사례:

- <b>정부 발행 자격증명(Government Credentials)</b>: 운전면허증, 여권 같은 정부 신분증의 유일성(uniqueness)을 인터넷 규모에서 확인하는 데 사용될 수 있음.

- <b>개인 정보 보호</b>: 자격증명 사용 시 필요한 정보(예: 18세 이상 여부)만 공개하고, 주소나 체중 같은 불필요한 정보는 노출하지 않음. 익명성은 온라인 상호작용에서 매우 중요함.

- <b>World ID Credentials 베타 출시</b>:

- AMPC의 규모와 익명성 덕분에 가능해졌으며, 프로토콜의 일부이므로 누구나 활용할 수 있음.

- 개인 정보 보호를 위해 여권 정보는 사용자 휴대폰에만 저장되며, 클라우드나 백엔드로 복사되지 않음. AMPC는 고유성 확인에만 사용됨.

- 미국 여권부터 지원하며, 몇 주 안에 다른 국가 여권도 추가될 예정임.

- Orb를 사용하지 못한 사용자의 네트워크 온보딩 가속화에도 활용될 수 있음 (여권 이미지와 휴대폰 사용자 이미지를 대조하여 확인).

- <b>World ID Deepface 발표</b>:

- 딥페이크(Deepfake) 문제 해결을 목표로 함 (예: CEO 사칭 금융 사기).

- World ID는 전 세계 규모에서 사람의 이미지를 캡처하고 서명하여 익명으로 신뢰할 수 있는 방식으로 제공하는 유일한 시스템임.

- World ID 2.0에 포함된 'Face Off' 기능은 Orb로 캡처된 이미지를 사용자 기기에 보관하며, 사용자와 기기 상호작용 시 본인 여부를 비교하여 인증함.

- 이 기술을 딥페이크 문제 해결에 적용함. 거래 시 상대방이 실제 본인인지 확인할 수 있도록 연결함.

- 현재 베타 개발 중이며, 곧 출시될 SDK 형태로 모든 애플리케이션에 통합 가능함. 기존 화상 통화 앱에서도 작동할 수 있는 베타 버전도 제공함.

- <b>디지털 신원 인프라 통합</b>:

- World ID를 다른 국가의 디지털 신원 인프라에 통합하는 방안을 논의 중임.

- 대만 등 두 국가에서 가장 활발한 대화가 진행 중이며, 입법가, 정책 입안자들과 협력하여 시민들에게 더 나은 서비스를 제공하고자 함.

## **World Chain: 프로토콜 확장**

- 구축 중인 것은 회사가 아닌 실제 '프로토콜'이며, 이는 기본 인프라가 분산화되어야 함을 의미함.

- 지난 1년 동안 이더리움 L2의 한계에 도달했으며, 네트워크 운영 비용이 매우 비싸졌음.

- <b>World Chain</b>을 발표함.

- '인간 증명' 시스템이 있기에 다른 블록체인이 할 수 없는 일을 할 수 있음.

- 실제 인간에게 무료 거래를 제공하여 훨씬 더 확장 가능하게 만듦.

- 자체 인프라를 필요에 맞게 조정할 수 있음.

- 오늘 World Chain이 출시됨.

- 출시 첫날부터 1,500만 명의 인간 사용자를 확보하여 세계 최대 블록체인 중 하나가 됨.

- 온램프(on-ramps), 오프램프(off-ramps) 등 사용자 및 개발자에게 필요한 모든 미션 크리티컬 통합이 이미 준비되어 있음.

- 이더리움에 의해 보안이 유지됨.

- 첫 10억 명의 사용자를 위해 Worldcoin 재단 보조금으로 5천만 WLD를 할당했음. 이는 개발자들이 함께 구축할 수 있는 큰 기회임.

## **World App 3.0: 인간을 위한 슈퍼 앱**

- 향후 몇 년 동안 많은 앱과 서비스가 Worldcoin 인프라를 활용하게 될 것이며, 사용자들은 이를 인지하지 못할 수도 있음.

- Worldcoin과의 통합을 훨씬 쉽게 만들기 위해 <b>Wallet Kit</b>을 SDK 형태로 올해 1분기에 출시할 예정임.

- World App은 지난 해 출시된 자체 클라이언트이며, 현재 세계에서 세 번째로 큰 암호화폐 지갑임.

- <b>World App 3.0</b>을 발표함.

- 일상생활에 Worldcoin 네트워크를 더욱 유용하게 만들기 위해 처음부터 재설계되었음.

- <b>World ID 통합</b>:

- 인터넷의 인간 신원 역할을 함.

- 본인 인증 상태를 확인하고, 인증되지 않았다면 앱을 통해 근처 Orb를 찾아 인증받을 수 있음.

- 새로운 World ID Credentials를 지원하여 여권이나 운전면허증 같은 정보를 World ID에 추가하여 익명성을 침해하지 않고 온라인에서 자신을 증명할 수 있음.

- 가을에는 World ID Deepface를 통해 자신이 주장하는 그 사람임을 증명할 수 있게 됨.

- <b>네트워크 내 다른 사람들과의 상호작용 용이성</b>:

- 사용자 이름으로 사람을 찾거나 전화 연락처를 연결하여 친구를 쉽게 찾고 송금할 수 있는 탭을 추가함.

- <b>지갑 기능 강화</b>:

- Worldcoin, 디지털 달러 및 기타 암호화 자산을 사용할 수 있음.

- 지난해 출시 이후 1,500억 건 이상의 검증과 15억 달러 이상의 거래량을 기록함 (Worldcoin 청구 제외).

- World Chain을 기반으로 구축되어 거래 처리 속도가 최대 90% 빨라짐.

- 전 세계 친구들에게 수수료 없이 즉시 송금하고, QR 코드로 상점에서 결제하거나, Shopify와 같은 온라인 상점에서 Worldcoin으로 결제할 수 있게 됨.

- <b>Vault(금고) 기능</b>: 미래를 위해 자금을 잠가두고 수익을 얻을 수 있으며, 자동으로 잔액이 증가하는 것을 볼 수 있음.

- <b>디자인 원칙</b>:

- 완전히 익명으로 사용할 수 있음.

- 중요한 데이터와 지갑 키는 사용자 기기에만 저장되어, 사용자가 완벽한 통제권을 가짐.

- 포괄적임: 수십 개 언어와 50개 이상의 통화를 지원하며, 최대 10년 된 휴대폰까지 지원함.

- <b>미니 앱(Mini Apps) 도입</b>:

- 앱 내에서 탐색하고 사용할 수 있는 타사 애플리케이션임.

- World ID, 지갑, 연락처와 깊이 통합되어 이전에 불가능했던 새로운 유형의 제품과 서비스를 제공함.

- <b>앱 스토어</b>: 카테고리별 탐색, 검색, 인기 차트 등을 제공함. World ID 기반으로 사용자 수 및 평판을 신뢰할 수 있음.

- <b>미니 앱 예시</b>:

- <b>Poll 앱</b>: 설문조사를 생성하고 소셜 미디어에 공유하며, 봇에 의해 왜곡되지 않은 실제 결과를 얻을 수 있음.

- <b>게임</b>: 인간 네트워크 기반 게임이 훨씬 재미있음 (예: Flappy Orb).

- <b>World Chat</b>: World 네트워크 기반의 암호화된 메시징 애플리케이션임. 사용자 이름이나 연락처를 통해 메시지를 보내고, 사진이나 음성 메모를 첨부할 수 있음.

- 지갑에 연결되어 있어 수수료 없이 전 세계로 즉시 송금할 수 있음.

- 채팅 상대방이 인증된 인간인지 (파란색 버블) 아닌지 (회색 버블) 알 수 있어 대화 시 주의할 필요가 있는지 판단하는 데 유용함.

- 미니 앱은 가까운 미래에 AI 에이전트가 이러한 작업을 위임받는 기반이 될 수 있음.

## **결론 및 리브랜딩**

- 오늘의 핵심 주제는 '규모(Scale)'였음 (새로운 Orb, World ID 3.0, World Chain, World App 3.0).

- 매우 시기적절하다고 판단하며, 최대한 빠르게 많은 사람에게 도달하는 것을 목표로 함.

- 기술 혁명과 플랫폼 전환의 시기에 살고 있으며, 이는 인류가 경험한 어떤 변화만큼이나 거대함.

- 프로젝트가 세계의 중요한 새로운 기술 인프라가 될 수 있기를 기대함.

- 'Worldcoin'이라는 이름이 더 이상 적합하지 않다고 판단하여, 오늘부터 'World'로 리브랜딩함.
