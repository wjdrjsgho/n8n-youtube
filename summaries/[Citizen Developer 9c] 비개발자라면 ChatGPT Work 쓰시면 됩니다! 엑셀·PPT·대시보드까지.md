## **ChatGPT Work 출시 및 기능 소개**

- 오픈AI에서 ChatGPT Work를 출시했음.

- 데이터를 분석하여 자동으로 정리하고 파일도 생성하는 기능을 제공함.

- 오늘 ChatGPT Work가 엑셀 데이터 분석, 리포트, PPT, 대시보드 제작 등 실용적인지 직접 활용해 볼 것임.

- 영상을 통해 본인만의 시스템을 구축하는 데 활용할 수 있음을 강조함.

## **ChatGPT Work의 개념 및 작동 방식**

- <strong>정의:</strong> 작업에 필요한 내용을 담은 폴더를 제공하면, 해당 파일에 접근하여 실제 업무를 대신해 주는 AI 에이전트임.

- 신입 직원에게 폴더를 주고 보고서 작성을 요청하는 것과 유사함.

- <strong>기존 Codex와의 유사점:</strong> 작동 방식은 기존 Codex와 유사함.

- Codex는 개발자용 코딩 툴에 가까웠음.

- ChatGPT Work는 비개발자용 지식 업무에 초점을 맞춘 툴임.

- <strong>Claude Co-work와의 유사점:</strong> Claude Co-work와도 유사한 종류의 업무를 시킬 수 있음.

- 둘 다 로컬 파일을 열고 다양한 앱과 연동하여 결과물을 생성할 수 있음.

- 본 영상에서 ChatGPT Work 사용법을 보여주고, Claude Co-work와 같은 프롬프트로 요청했을 때의 결과물을 비교할 것임.

- 시청자가 본인의 업무 워크플로우에 어떤 툴이 더 적합한지 판단해 볼 수 있도록 안내함.

## **실습 예시 및 준비된 데이터**

- <strong>가상 시나리오:</strong> 가상의 동네 카페 '행복 카페 성수점'을 운영한다고 가정함.

- <strong>데이터 준비:</strong> 2분기 매출 데이터를 입력한 샘플 시트를 제작했음.

- 매출 상세 데이터 포함.

- 메뉴 데이터, 매입 내역, 재고 실사, 고객 후기, 인력 운영, 마케팅 캠페인, KPI 목표, 데이터 사전 시트들을 생성했음.

- 참고 자료로 운영 브리프 및 카페 로고를 샘플로 준비했음.

- <strong>데이터 규모:</strong> 매출 데이터는 22,000건이 넘는 비교적 큰 규모의 엑셀 데이터임.

- 해당 엑셀 파일을 통해 ChatGPT Work의 실제 작업 능력을 확인할 것임.

## **ChatGPT Work로 제작할 결과물**

- 총 세 가지 결과물을 제작할 것임.

- <strong>엑셀 데이터 분석:</strong> 제공된 파일 기반으로 엑셀 데이터 분석을 요청하고, 인사이트 문서도 별도로 생성하도록 요청할 것임.

- <strong>PPT 슬라이드 보고서:</strong> 데이터 분석 인사이트를 보기 좋게 정리한 PPT 슬라이드를 제작하도록 할 것임.

- <strong>대시보드 제작 및 배포:</strong> ChatGPT의 'Sites' 최신 플러그인을 활용하여 대시보드를 제작하고, 이를 외부 공개 가능한 URL로 배포하도록 요청할 것임.

## **ChatGPT Work 활용 방법**

- <strong>접근 방식:</strong>

- 브라우저(chatgpt.com)의 Work 탭을 통해서도 활용할 수 있음.

- 하지만 컴퓨터 내 폴더와 연동하여 사용하려면 데스크톱 앱을 설치해야 함.

- <strong>데스크톱 앱 인터페이스:</strong> 데스크톱 앱 설치 시 왼쪽 상단에 ChatGPT Work와 Codex를 전환할 수 있는 탭이 생성됨.

- Codex는 앱 제작, 디버그, 배포에 추천됨.

- ChatGPT Work는 지식 업무 활용에 가이드됨. (비개발자에게 더 적합함)

- <strong>모델 및 설정:</strong>

- 모델: 최신 모델인 GPT-4o를 활용할 것임.

- 레벨: Extra High로 설정할 것임.

- 스피드: Fast로 설정할 것임.

- <strong>프로젝트 폴더 선택:</strong> 샘플 데이터가 포함된 폴더를 지정하여 프로젝트를 시작할 것임.

## **엑셀 데이터 분석 요청 및 결과**

- <strong>프롬프트 작성 방식:</strong>

- 어떤 데이터가 있는지 알려주고, 데이터 구조와 발견한 문제에 맞게 직접 시트 구성이나 차트를 설계하도록 요청함.

- 오픈AI 공식 가이드에 따라 상세 단계를 일일이 적기보다 필요한 결과부터 말하고 접근 방법은 ChatGPT에 위임하는 방식을 따름.

- 반드시 포함해야 할 내용만 추가함.

- <strong>작업 진행 과정:</strong>

- ChatGPT Work는 먼저 작업 계획을 제시함.

- 스프레드시트 스킬을 활용하여 데이터를 읽어 들임. (데이터 양이 많아 시간이 소요됨)

- 5단계 진행 계획을 알려줌: 파일 구조 체크 -> 데이터 완전성, 중복, 유효성, 정합성 점검 -> 데이터 분석 -> 수식 기반 엑셀 및 핵심 인사이트 마크다운 생성 -> 품질 검증.

- <strong>분석 결과:</strong>

- 약 25~30분 소요 후 결과물이 생성됨.

- 분석 결과와 핵심 인사이트 두 가지로 나눠서 생성을 완료함.

- 주요 결론 제시: 목표의 97.2% 달성, 순매출 전월 대비 증가, 공헌 이익 1.9% 감소, 관리 가능 이익 -244만원 등.

- <strong>핵심 인사이트 (마크다운 파일):</strong>

- 한 줄 결론과 데이터 품질 판정 내용을 포함함.

- 제공된 샘플 데이터에서 발생한 문제점(데이터 정합성 등)을 지적하여, 실제 데이터 문제가 있을 경우 수정 후 재분석 요청이 가능함을 시사함.

- 경영 요약 및 수익성, 주요 원인, 7월 실행 액션, 검산 결과 등을 잘 정리했음.

- <strong>엑셀 파일 결과:</strong>

- 엑셀과 연동하여 확인 시, '경영 요약' 시트에 대시보드 형태로 차트와 함께 깔끔하게 정리되어 있음.

- 대표 결론, 월별 순매출, 공헌 이익을 차트로 잘 표기함.

- KPI, 수익성, 원인 분석(채널, 메뉴, 서비스, 인력, 재고, 공급, 마케팅), 데이터 품질 이슈 등 다양한 분석 시트를 생성함.

- 7월 액션도 정리되어 있음.

- 로데이터 기반으로 수식을 적용하여 분석했기 때문에, 수식을 확인하며 데이터 검증이 용이함.

- <strong>결과물 수정 요청:</strong>

- 7월 실행 액션 부분만 '무엇을, 왜, 언제까지, 무엇으로 성공을 확인하나'의 네 가지 섹션으로 다시 정리해 달라고 요청했음.

- ChatGPT Work가 해당 부분만 수정하여 요청한 형태로 가공된 최종 버전을 생성함.

- 초안이 완벽하지 않아도 피드백을 통해 원하는 형태로 수정 가능함을 보여줌.

## **PPT 슬라이드 보고서 요청 및 결과**

- <strong>프롬프트 작성 방식:</strong>

- 분석 결과와 인사이트를 바탕으로 8~10장 분량의 PPT를 제작하도록 요청함.

- 반드시 장표에서 답변해야 할 내용들을 강조함.

- 제공된 브랜드 가이드와 카페 로고를 반영해 달라고 요청함.

- <strong>작업 진행 및 결과:</strong>

- 약 17분 30초 소요 후 10장 분량의 경영진용 리포트 PPT를 생성함.

- PPT 파일 확인 결과, 샘플 로고가 반영되어 스타일에 맞게 잘 제작됨.

- 내용 구성:

- 리포트 요약 슬라이드 (순매출, 공헌 이익률, 주문수, 관리 가능 이익, 경영 판단 등 깔끔하게 정리됨).

- 바 차트 포함 월별 순매출 변화, 공헌 이익률 하락 메시지, 배달 성장 및 마진 이슈 등 주요 이슈를 잘 짚어줌.

- 전반적으로 차트까지 포함하여 깔끔하게 제작되었으며, 이슈와 액션 플랜까지 잘 정리함.

## **대시보드 제작 및 배포 요청 (Sites 기능 활용)**

- <strong>프롬프트 작성 방식:</strong>

- '@사이트' 기능을 활용하여 대시보드 제작을 요청함.

- Sites 기능은 대시보드 제작뿐만 아니라 URL로 배포하여 공유할 수 있는 점을 강조함.

- <strong>작업 진행 및 결과:</strong>

- 대시보드 제작 완료 후, ChatGPT 계정으로 로그인하여 확인할 수 있게 연결됨.

- 대시보드 구성:

- 순매출, 공헌 이익률, 주문수, 관리 가능 이익, 객단가, 공헌 이익 금액 등을 보여줌.

- KPI, 공헌 이익률 추이, 수익성 하락 주요 원인, 월별 폐기 원가 및 목표 등을 반영하여 깔끔하게 제작됨.

- <strong>추가 기능:</strong>

- PDF나 HTML로 내보내기가 가능함.

- 'Edit' 버튼을 눌러 카드 위치 이동 등 대시보드 구성을 수정할 수 있으며, 즉시 반영됨.

- 'Share' 기능을 통해 'anyone on the internet'으로 설정하여 손쉽게 URL로 공유할 수 있음.

## **ChatGPT Work와 Claude Co-work 결과물 비교**

- <strong>엑셀 분석 결과 비교:</strong>

- <strong>Claude Co-work (왼쪽):</strong>

- 안내 시트(설명서)를 먼저 제작하여 시트 구성을 설명해 주는 점이 의미 있음.

- 컬럼 사이즈 가독성 조정은 아쉬운 부분임.

- 경영 요약은 한 줄 결론과 함께 7가지 질문에 대한 답변을 구체적으로 해 줌.

- 섹션별로 세부 분석을 '원샷'으로 제공하는 경향이 있음.

- 세부적인 내용들을 각각 해석해 주는 점이 깊이 있게 느껴짐.

- <strong>ChatGPT Work (오른쪽):</strong>

- 경영 요약을 대표 수치와 결론, 차트와 함께 좀 더 깔끔하게 요약한 느낌임.

- 분석의 깊이는 Claude Co-work보다 다소 간단하게 느껴질 수 있음.

- 세부적인 분석 내용들을 세 줄 요약 형태로 제공하는 경향이 있음.

- <strong>총평:</strong> 둘 다 잘 분석했으나, Claude Co-work가 섹션별 세부 분석에서 깊이가 더 있는 것처럼 보임. ChatGPT Work도 나쁘지 않으나 간결한 느낌을 줌.

- <strong>PPT 보고서 비교:</strong>

- <strong>Claude Co-work (왼쪽):</strong>

- 표지는 비슷함.

- 요약은 '매출은 늘었는데 남는 게 줄었다', '매출 목표에 근접했지만 이익 질이 약해졌다' 등으로 표현함.

- 매출 비교 시 목표와 함께 보여주는 부분이 더 좋게 평가됨.

- 공헌 이익률 하락에 대한 세부적인 비중을 보여주는 점이 나음.

- 액션 플랜 마지막에 '계산 가능한 순 효과'를 제시하여 특정 액션 시 얻을 수 있는 효과를 요약하는 점이 좋음.

- 슬라이드 구성이 다채로운 부분이 있음.

- <strong>ChatGPT Work (오른쪽):</strong>

- 표지는 비슷함.

- 요약은 '경영 판단' 섹션 구성이 좋게 평가됨.

- 공헌 이익률 하락에 포커스를 둔 설명도 나쁘지 않음.

- 액션 플랜을 잘 정리했음.

- 디자인 측면에서 로고를 우측 상단에 반영하고 풋노트를 넣는 등 괜찮은 부분이 있음.

- PPT 퀄리티가 예전보다 많이 따라잡았음.

- <strong>총평:</strong> PPT 퀄리티는 거의 비슷하나, 디자인적인 측면이나 슬라이드 구성의 다채로움, 그리고 마지막 결론 부분의 순 효과 제시에서 Claude Co-work가 조금 더 낫다는 평을 받았음.

- <strong>대시보드 비교:</strong>

- <strong>Claude Co-work (왼쪽):</strong>

- 로고가 반영되는 등 디자인이 제대로 반영됨.

- 필터링 기능 제공 등 사용자 인터랙션 측면이 좋음.

- 내용 요약 카드와 차트(공헌 이익률 등)를 이쁘게 제작함.

- 질문 내용에 대해 비교하며 제작을 잘 해줌.

- <strong>ChatGPT Work (오른쪽):</strong>

- 디자인적인 측면(로고 반영 등)에서 아쉬움을 보임.

- 질문한 내용들에 대한 제작은 잘 해줌.

- 'Edit' 기능을 통해 대시보드 구성을 바로 수정할 수 있는 장점이 있음.

- <strong>총평:</strong> 원샷 퀄리티 면에서는 Claude Co-work가 디자인과 인터랙션에서 조금 더 나았음. 하지만 ChatGPT Work는 수정 및 손쉬운 공유 기능에서 장점이 있음.

## **ChatGPT Work의 장점 및 활용 팁**

- <strong>사용량:</strong> ChatGPT Work는 사용량 제한이 넉넉한 편임 (거의 달지 않았음).

- <strong>피드백 활용:</strong> ChatGPT는 디렉션을 잘 따르는 모델이므로, 원하는 부분에 대한 피드백을 주고받으면 Claude Co-work와 충분히 비슷한 퀄리티의 결과물을 만들 수 있음.

- <strong>초안 퀄리티:</strong> 자율적으로 분석 요청을 해도 어느 정도 퀄리티 높은 초안을 제작해 주는 점이 인상적임.

- <strong>Sites 기능의 유용성:</strong> Sites 기능을 활용하여 손쉽게 대시보드를 배포하고 활용할 수 있는 것이 큰 장점임.

- <strong>전반적인 분석 깊이 및 대시보드 퀄리티:</strong> 원샷 기준으로 Claude Co-work가 다소 나은 편이지만, ChatGPT Work의 피드백 반영 능력과 후한 사용량으로 충분히 극복 가능한 수준의 퀄리티를 보여줌.
