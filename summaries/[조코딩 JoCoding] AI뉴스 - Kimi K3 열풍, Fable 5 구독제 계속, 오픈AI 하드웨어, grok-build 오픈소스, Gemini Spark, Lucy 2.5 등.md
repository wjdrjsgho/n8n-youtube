## **키미 K3 모델 공개 및 성능**

- 키미 K3 모델이 공개됨. 곧 오픈 웨이트로 풀릴 예정임.

- 이전 키미 모델들이 꽤 괜찮은 평가를 받았음.

- 
        이번 모델의 가장 놀라운 점은 프론티어급 성능을 보인다는 점임.

- 클로드 3.5나 GPT-4o와 필적하는 수준의 성능을 보인다고 화제가 됨.

- 특히 코딩 성능에서 굉장한 우위를 보임.

- 이미 GPT-4.8 같은 모델들을 뛰어넘었음.

- 
        프론티어 코드 아레나에서 1위를 차지함.

- 클로드 3.5, GPT-4o 솔과 경쟁하여 투표를 통해 1등을 함.

- 이는 중국 오픈 웨이트 모델이 미국 모델을 처음 앞지른 역사적인 사건으로 평가됨.

- 딥시크 사태 때도 화제가 되었으나 당시에는 미국 모델을 뛰어넘지 못했으나, 키미 K3는 이를 가능하게 함.

- 
        프론트엔드 코드 아레나 비교 결과

- 동일한 프롬프트로 키미 K3와 클로드 3.5를 비교함.

- 키미 K3가 꽤 잘 만들어진 결과물을 보이며, 더 디테일하고 임팩트 있게 나옴.

- 이런 모델을 페이블 3.5를 앞선 상황으로, 서버에서 무료로 돌릴 수 있다는 점이 강점임.

- 데이터 센터가 있는 집이라면 개인적으로도 돌릴 수 있음.

## **키미 K3 상세 스펙 및 벤치마크**

- 
        모델 스펙

- 2.8조 파라미터 규모임.

- 일반적인 개인 장비(RTX 5090 등)로는 쉽게 돌릴 수 없는 사이즈임.

- 100만 토큰 컨텍스트를 제공함.

- 
        공식 홈페이지 벤치마크 점수 공개

- 코딩 영역에서 DeepSeek, GPT-4o, 클로드 3.5 다음으로 67.5점으로 3위를 기록함.

- 놀랍게도 실제 테스트에서는 69점을 기록했으나, 겸손하게 1% 낮춰서 공개함 (클로드 3.5는 70점).

- 이는 키미 K3가 코딩을 매우 잘한다는 것을 의미함.

- 그 외 영역들도 꽤 괜찮은 수준의 성적을 보이며, 종합 점수에서 오픈 웨이트 모델 중 독보적인 위치를 차지함.

- 
        사이버 보안 성능 평가

- 버셀(Vercel) CEO가 내부 평가 결과, 키미 K3가 사이버 보안에서 최상의 수준이라고 언급함.

- 클로드와 같은 모델은 거부하거나, GPT-4o와 같은 모델은 비용이 많이 들지만, 키미 K3는 보안 분야에서 활용 가능하며 오픈 웨이트 모델이기에 안전 해제 후 무제한 사용 가능함.

- 보안 분야에서 가장 좋은 모델이 될 수도 있다고 평가됨.

## **기업 가치 및 추가 벤치마크**

- 키미 K3를 만든 기업의 기업 가치는 약 200억 달러로, OpenAI나 Anthropic에 비해 현저히 낮음.

- 낮은 기업 가치에도 불구하고 프론티어 랩에 준하는 모델을 공개한 점이 놀라움.

- 
        하비 랩(Harvy Lab) 법률 관련 벤치마크에서 1위를 차지함.

- 클로드 3.5보다 거의 10% 넘게, 약 2배 가까운 점수를 달성함.

- 이는 키미 K3가 법률 분야에서도 매우 뛰어난 모델임을 시사함.

## **키미 K3 API 가격 및 활용 사례**

- 
        오픈 웨이트 공개일 및 API 가격

- 7월 27일부터 다운로드 가능함.

- API 가격은 입력 100만 토큰당 3달러, 캐시 0.3달러, 출력 15달러임.

- GPT-4o 솔(입력 5달러, 출력 30달러)이나 클로드 3.5(50달러 기준)보다 훨씬 저렴한 수준임 (거의 반값 또는 30% 가격).

- 출론 시 실제 토큰 소모량을 고려하면 GPT-4o와 큰 차이는 없지만, 클로드 3.5에 비하면 많이 저렴함.

- 
        활용 사례 시연

- 완전 기능 macOS 스타일 시스템을 키미 K3로 제작함.

- 브라우저, 메모장, 게임 플레이 등 동물의 숲과 유사한 UI를 구현함.

- 
        투자 영역에서의 관점 (세미애널리시스)

- 일부 사람들의 잘못된 인식과 달리, 키미 K3는 엔비디아, HBM, 디램, 네트워킹에 부정적이지 않음.

- 오히려 2.8조 파라미터의 가중치 저장 및 추론을 위해 대규모 확장 도메인이 필요하여, 엔비디아 GPU, HBM, 디램 등의 수요를 증가시킬 수 있음.

- 미국 모델들이 높은 가격에 경쟁력을 유지할 수 있을지는 또 다른 문제임.

## **Grok Build CLI 보안 이슈 및 오픈 소스 공개**

- 
        보안 이슈 발생

- Grok Build CLI가 개발자의 의도와 달리 전체 Git 저장소(히스토리, AI가 읽지도 않는 파일 포함)를 서버에 업로드하고 있었음.

- 사용자가 질문할 때만 파일을 보내는 것이 아니라, 불필요한 파일까지 전송되어 보안적으로 중요한 파일(API 키 등)이 노출될 위험이 있었음.

- 약 26,000배 이상의 데이터가 불필요하게 전송됨.

- 개인 정보 설정으로도 이를 막을 수 없었음.

- 
        XAI의 대응

- 논란 이후 서버 업로드 기능을 비활성화함.

- 일론 머스크는 직접 업로드된 데이터 모두 삭제하겠다고 언급함.

- 
        오픈 소스 공개

- 보안 문제를 의식하여 Grok Build CLI를 오픈 소스로 공개함.

- 개발 하네스가 어떻게 구성되어 있고, 어떤 파일을 서버로 가져가는지 투명하게 공개함.

- GitHub에 Grok Build가 올라와 모델이 어떤 파일을 활용하고 어떻게 동작하는지 투명하게 확인할 수 있음.

## **새로운 오픈 웨이트 모델 및 AI 기술 발전**

- 
        프리즘 ML(Prism ML)의 본사이 AI 2.7B 모델

- 휴대폰에서 실행되는 최초의 2.7B 클래스 모델임.

- Q3 3.6 27B 모델을 1비트 양자화로 변환하여 3.9GB로 줄임.

- 1비트 양자화에도 불구하고 성능 저하가 크지 않아(종합 점수 85점에서 76점), 스마트폰에서도 76점 수준의 모델을 사용할 수 있게 됨.

- 
        싱킹 머신즈(Thinking Machines)의 인클링(Inkling)

- 미라 무라티(Mira Murati)가 창업한 회사에서 공개한 새로운 오픈 웨이트 모델임.

- 허깅 페이스에서 다운로드 가능함.

- 다른 오픈 웨이트 모델들과 비교했을 때 나쁘지 않은 성능을 보이지만, GPT-4o 솔 같은 최상위 모델과는 다소 격차가 있음.

- 
        모바일 원(Mobile One) 영상 생성 모델

- 스마트폰에서도 구동 가능한 사이즈의 영상 생성 모델임 (1.25B 파라미터).

- 5B 모델과 유사하거나 더 나은 품질을 보임.

- 5초 분량의 480x832 해상도 영상을 초당 16프레임으로 20초 이내에 생성함.

- 오픈 웨이트로 공개됨.

- 
        완(Wan) 스트리머의 0.3 버전

- 실시간 스트리밍 AI 인플루언서 생성 모델의 다음 버전임.

- 동작까지도 실시간 상호작용이 가능하여, 원하는 동작을 말하면 거의 실시간으로 반응하여 동작을 생성함.

- 
        루시 2.5(Lucy 2.5) 실시간 비디오-투-비디오 교체 기술

- 실시간 라이브 스트림에서 영상의 인물을 AI로 변형할 수 있는 기술임.

- 가상의 AI 캐릭터(예: 가상 남자친구)를 만들어 실시간으로 상호작용하는 콘텐츠 제작에 활용 가능함.

- 스트리밍, 이커머스 등 다양한 분야에서 활용 가능성이 있음.

- 
        엔비디아(Nvidia)의 대화형 인간 동작 생성 오픈 소스 모델

- 자연어 프롬프트로 인간의 동작을 생성할 수 있는 모델임.

- "a person is waving", "jump backward", "chicken dance" 등 연속적인 동작을 생성함.

- 로봇의 움직임을 텍스트 기반으로 제어하는 데 확장 적용될 수 있음.

- 오픈 웨이트로 공개되어 로봇 분야 종사자들이 활용할 수 있음.

- 
        완서(Wanser) 음악에 맞춰 춤을 만들어내는 모델

- 음악에 맞춰 자연스러운 춤을 생성함.

- 중국 고전 무용, K-팝 댄스 등 다양한 댄스 종류를 그럴싸하게 만들어냄.

- 쇼츠(Shorts) 등 짧은 영상 콘텐츠 제작에 활용될 수 있음.

## **로봇 기술 발전**

- 
        중국 엔딩AI(Engine AI) 로봇 기업의 로봇 전투 시연

- 로봇들이 격렬한 돌려차기 등 화려한 발차기로 싸우는 모습을 공개함.

- 목이 떨어져 나가도 계속 싸우는 등 높은 내구성과 자율성을 보임.

- 일론 머스크도 로봇 싸움에 흥미를 표했으며, 새로운 스포츠 종목 탄생 가능성이 있음.

- 
        엔딩AI의 자율 비행 호버보드 로봇

- 로봇이 호버보드를 타는 모습을 시연함.

- 안전 테스트를 위해 로봇이 탑승한 것이며, 향후 인간도 탈 수 있을 것으로 예상됨.

- 
        선데이 로보틱스(Sunday Robotics)의 액트-2(Act-2) 공개

- 방범위한 일반화와 높은 신뢰성을 통합하는 최초의 로보틱스 모델임.

- 단일 미세 조정 예시만으로 새로운 행동을 일반화하여 가르칠 수 있음.

- 학습에 사용되지 않은 행동도 99%의 성공률로 수행할 수 있음.

- 빨래 개기, 커피 따르기 등 일상적인 작업을 보고 바로 배울 수 있음.

- 빨래를 아무렇게나 던져줘도 스스로 파인 튜닝하여 잘 접는 모습을 보여줌.

## **AI 기업 로고 디자인 트렌드**

- 
        AI 기업 로고들의 유사성 지적

- 많은 AI 기업 로고들(특히 Anthropic)이 비슷하고 '학문'처럼 보이는 경향이 있음.

- 이는 업계의 모방과 위험 회피적인 디자인 선택, 그리고 혁신을 강조하려는 기업들이 오히려 서로 닮게 만드는 경향 때문임.
