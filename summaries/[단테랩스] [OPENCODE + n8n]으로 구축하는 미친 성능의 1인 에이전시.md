## **영상 소개 및 새해 소식**

- 지난해는 <strong>에이전트 시대의 원년</strong>이었음.

- NAN, 클로드, MCP 등 AI 자동화 기술이 쏟아져 나왔음.

- 2026년은 이 에이전트들이 <strong>초개인화 시대</strong>로 넘어가는 해가 될 것이라 예상함.

- AI 기술이 얼리 어댑터들의 전유물인 '캐즘'이 깨져 일반 사무직들도 개인화된 에이전트를 직접 만드는 것이 일상화될 것이라 전망함.

- 새해를 맞아 두 가지 소식을 준비했음.

- AI 에이전트와 자동화에 관한 열린 지식의 장인 <strong>단테랩스 지식 커뮤니티</strong>를 오픈함.

- 현재 집필 중인 <strong>N8N 업무 자동화 도서 제목 공모 이벤트</strong>를 진행함.

- 총 50명을 추첨하여 1만 원 상당의 기프티콘을 지급함.

- 이벤트 참여 및 커뮤니티 입장 방법은 영상 마지막과 설명란, 고정 댓글에 자세히 안내함.

- 오늘 클로드 코드의 대항마인 <strong>오픈 코드(Open-Code)</strong>를 N8N에 통합하여 높은 수준의 마케팅 비즈니스를 1인이 구현하는 절차와 가능성을 보여줄 예정임.

- 오픈 코드로 에이전트 운영 비용을 거의 제로에 가깝게 만들 수 있음을 언급함.

## **오픈 코드(Open-Code)의 이해 및 설치**

- <strong>오픈 코드 소개</strong>

- 클로드 코드의 대항마로 등장한 코드 어시스턴트 툴임.

- 터미널 창에서 LLM을 사용할 수 있으며, 클로드 코드와 유사함.

- <strong>클로드 코드와의 주요 차이점</strong>

- 오픈 소스이며 커뮤니티에 의해 계속 성장함.

- 다양한 LLM 서비스 제공자들을 연결하여 여러 모델을 사용할 수 있음 (API 키 또는 OAuth 인증 방식).

- 클로드 코드는 Anthropic 모델만 사용 가능함.

- 무료 모델도 많이 제공하여 완전 무료로 에이전트 비즈니스 업무를 수행할 수 있음을 강조함.

- 복잡하지 않은 콘텐츠 생성 작업 시 클로드 코드 대비 매우 비용 효율적임.

- <strong>오픈 코드 설치 과정 (Windows 기준)</strong>

- 오픈 코드 웹사이트(open-code.ai)에 접속하여 설치 명령어를 확인했음.

- CURL 대신 NPM(Node Package Manager)을 통해 설치함 (더 안정적임).

- NPM은 Node.js가 설치되어 있어야 하며, Node.js 설치 튜토리얼을 영상 하단에 안내함.

- 제공된 명령어를 터미널 창에 복사 후 붙여넣기 실행함.

- `open-code -v` 명령어로 버전이 출력되면 설치가 완료된 것임.

- <strong>오픈 코드 인증 및 LLM 연결</strong>

- <strong>Open-Code Geni (무료 모델) 연결</strong>

- 오픈 코드 웹사이트의 Geni 서비스에 접속하여 로그인하고 무료 플랜으로 이용함.

- API 키를 발급받아 복사함.

- 터미널에서 `open-code login` 입력 후 목록에서 'Open-Code Geni'를 선택함.

- 발급받은 API 키를 붙여넣어 연결함.

- <strong>Google LLM (Gemini) 연결 (Antigravity 이용)</strong>

- API 키 방식은 비용이 발생할 수 있어 Antigravity를 통한 OAuth 인증 방식을 사용함.

- <strong>Antigravity 설치</strong>

- Google/download에서 OS에 맞는 Antigravity를 다운로드 및 설치함.

- 설치 완료 후 실행하여 'Login with Google' 버튼으로 로그인하여 Google 구독 계정을 연결함.

- <strong>Antigravity 플러그인 설치 (Open-Code 통합)</strong>

- Open-Code 공식 문서의 플러그인 섹션에서 'Open-Code Antigravity OAuth' 플러그인을 찾음.

- 수동 설치 방법으로 진행함.

- 사용자 루트 디렉토리 내 `.config/open-code/` 경로에 `open-code.json` 파일을 생성함.

- 제공된 JSON 내용을 복사하여 파일에 붙여넣고 저장함.

- Antigravity 터미널 창을 열어 `open-code login` 입력 후 'Google'을 선택함.

- 'Antigravity의 OAuth 인증 방식'을 선택 후 프로젝트 ID는 비워둠.

- 리다이렉션된 로그인 페이지에서 Google 계정으로 로그인 후 주소를 복사하여 터미널에 붙여넣음.

- 'N'을 눌러 어카운트 취소 시 로그인 완료됨. 이제 오픈 코드에서 Google Gemini 모델 사용 가능함.

- <strong>GLM 4.7 (가성비 모델) 연결</strong>

- GLM 4.7은 Claude와 견줄만한 성능에 비해 가격이 매우 저렴한 모델로, 특히 복잡하지 않은 비즈니스 업무에 추천함.

- z.ai 웹사이트에 접속하여 가입 후 API 섹션에서 API 키를 발급받음.

- 터미널에서 `open-code login` 입력 후 'Z AI Coding Plan'을 검색하여 선택함.

- API 키를 입력하여 연결함.

- 총 세 개의 서비스 제공자(Open-Code Geni, Google Gemini, Z AI GLM 4.7)를 오픈 코드에 연결했음을 확인함.

## **오픈 코드 플러그인 및 환경 설정**

- <strong>Oh My Open-Code 플러그인 설치</strong>

- 클로드 코드의 스킬, 에이전트, 커맨드 등 모든 명령어들을 오픈 코드에서 사용할 수 있도록 연결하는 기능이 있음.

- 오픈 코드의 에이전트 기능을 확장하여 전문 하위 에이전트들을 통해 더 에이전틱한 업무 환경을 만들어줌.

- 한국인 개발자가 개발했으며, 병렬 처리 등 퍼포먼스 향상에 기여함.

- ohmyopen-code.com 웹사이트에서 설치 방법을 확인함.

- MPX를 통한 Node.js 설치 방법을 사용함.

- Antigravity 터미널 창에 `npx @dantelabs-agentic-school/oh-my-open-code install` 명령어를 입력 후 실행함.

- 설치 과정에서 Claude, ChatGPT, Google Gemini 구독 여부를 물어보며, 구독 상황에 맞춰 응답함.

- `open-code.json` 파일에 플러그인이 자동으로 설치된 것을 확인함.

- 시지푸스, 라이브러리안 등 다양한 에이전트 퍼스나가 마련되어 있고, 각 에이전트마다 모델 설정이 가능함을 언급함.

- <strong>오픈 코드 실행 및 GUI 환경 설정</strong>

- 터미널에 `open-code`를 입력하여 TUI(Text User Interface) 환경으로 오픈 코드를 실행함.

- Oh My Open-Code 플러그인이 잘 설치되었다면 에이전트가 <strong>시지푸스(Sisyphus)</strong>로, 모델은 GLM 4.7로 기본 선택되어 있음을 확인함.

- 터미널 창의 편의성을 위해 <strong>Open-Chamber 익스텐션</strong>을 소개함.

- Antigravity의 익스텐션 아이콘을 눌러 'Open-Chamber' 플러그인을 설치함.

- 이 확장은 오픈 코드를 GUI 환경에서 사용할 수 있도록 지원함.

- 설치 완료 후 프로젝트 작업을 위한 준비가 모두 완성됨.

## **가상 브랜드 마케팅 프로젝트 진행**

- <strong>프로젝트 폴더 생성 및 Open-Chamber 세션 시작</strong>

- Antigravity에서 작업할 폴더('단테 커피' 가상 브랜드 폴더)를 생성하고 프로젝트를 열었음.

- Open-Chamber 아이콘을 눌러 새 세션을 시작하고, 'Open Session in Editor'를 눌러 우측 패널에 세션 창을 생성함.

- 세션 에이전트가 시지푸스(Sisyphus)로, 모델이 빅 피클(무료 모델)로 설정되어 있는지 확인함.

- <strong>클로드 코드 에이전틱 환경 이해 및 오픈 코드 적용</strong>

- 클로드 코드는 메인 에이전트가 서브 에이전트를 호출하여 독립적으로 동작하게 함.

- 서브 에이전트는 입출력 내용만 공유하여 서브 태스크 위임이나 병렬 실행에 유용함.

- 스킬은 정해진 워크플로 실행에, 에이전트는 지능적 판단이 필요한 창발성 업무에 적합함.

- 모두 `.open-code/claude/` 하위 폴더(skills, commands, agents)에 마크다운 파일로 관리됨.

- 스킬 파일은 반드시 `skill.md`를 포함해야 하며, 프런트 매터(Front Matter)로 이름과 설명을 명시함.

- 커맨드는 트리거 역할을 하며, 에이전트는 자동 생성 기능도 있으나 수동으로도 마크다운 파일을 만들어 관리함.

- <strong>마케팅 전문 에이전트 다운로드</strong>

- 미리 만들어 둔 에이전트, 커맨드, 스킬을 다운로드하기 위해 터미널에 `npx @dantelabs-agentic-school/agentic-skill install` 명령어를 입력했음.

- 다운로드된 에이전트들은 마케팅 비즈니스 처리를 위한 스킬, 에이전트, 커맨드들로 구성되어 있음을 확인했음.

- 실제 업무에 활용 가능하도록 완성도를 높였으며, 지속적인 개선이 필요함을 언급함.

- <strong>에이전틱 마케팅 파이프라인 시나리오</strong>

- 시장 리서치 (시장 규모, 트렌드, 경쟁 환경 분석)

- 브랜드 분석 (SWOT, 경쟁사 포지션 분석)

- 고객 세그멘테이션 (주요/보조 고객 정의)

- 페르소나 정의 (마케팅 대상 프로파일링)

- 채널 전략 수립 (마케팅/홍보 전략)

- 콘텐츠 제작 (다양한 단계 포함)

- 이 과정들은 에이전트에 매핑되어 순차적 또는 병렬로 진행될 수 있으며, 전문 위임을 통해 최종 홍보물과 전략서를 만들 수 있음.

- <strong>가상 브랜드 '단테 커피' 소개 및 시나리오 다운로드</strong>

- `npx @dantelabs-agentic-school/marketing-sample install` 명령어를 통해 마케팅 샘플을 다운로드했음.

- 다운로드된 '단테 커피 브랜드' 문서는 브랜드 개요, 시장 정보, 제품 라인업을 정리한 가상 브랜드 문서임.

- '시나리오' 문서는 수행할 작업들을 에이전트들에 매핑하여 정리해 놓았음.

- <strong>오픈 코드 프로젝트 실행 (단계별 마케팅 파이프라인)</strong>

- <strong>초기화 및 한글 지침 설정</strong>

- `@파일 선택` 명령어로 '단테 커피 브랜드' 문서를 참조하여 읽고 파악하도록 요청함.

- 오픈 코드의 답변이 영문이므로, `/init` 명령어로 프로젝트 초기화를 진행함.

- '모든 작업을 한글로 하도록 지침에 명시해 줘'라고 입력하여 `agent.md` 파일을 생성하고 전체 코드 베이스를 이해하게 함. 이는 장기 기억 메모리로 활용되어 세션이 새로 시작돼도 규칙 아래에서 움직이게 됨.

- `/new` 명령어로 컨텍스트를 새로 초기화해도 `agent.md` 파일 덕분에 전체 내용을 이해할 수 있음.

- <strong>시장 리서치</strong>

- `/m:market` 커맨드를 선택하고, '단테 커피 브랜드' 문서를 참조하여 요청함.

- 첫 번째 보고서 내용을 작성함.

- `/m:landscape` 커맨드를 선택하고, 추가적으로 "완성된 보고서는 규칙에 따라 저장해 줘"라고 요청함.

- 'reports' 폴더에 경쟁사 분석 및 산업 구조 분석 보고서가 파일로 저장됨.

- 커맨드가 에이전트와 스킬을 작동시키며, 에이전트는 스킬보다 더 고민하고 추론, 피드백이 가능한 장치임을 언급함.

- 에이전트가 다음 작업을 제안하며, 이는 시나리오를 이해하고 있기 때문임.

- <strong>브랜드 분석</strong>

- `/m:brand` 커맨드를 선택하고, 브랜드 브리프 문서를 참조하여 브랜드 분석을 요청했음.

- 'brand-analysis' 폴더에 SWOT 분석, 경쟁사 포지션 분석 등 마케팅 전략 프레임워크에 맞춰 고품질 문서가 자동으로 작성되고 저장됨.

- 에이전트와 스킬을 통해 체계적이고 일관된 품질로 문서 생성이 가능함을 강조함.

- <strong>고객 세분화 및 페르소나 정의 (병렬 처리)</strong>

- 명시적인 커맨드 대신 "고객 세분화와 레포트 생성을 전문 에이전트에 맡겨서 병렬로 진행해 줘"라고 요청했음.

- 시지푸스 에이전트가 클로드 코드용으로 작성된 에이전트와 스킬을 읽어 하위 에이전트를 할당하고 병렬 작업을 진행함.

- 두 개의 새로운 세션이 생성되어 작업이 병렬로 처리됨.

- 작업 완료 후 'marketing' 하위 폴더에 페르소나('김지현' 인물) 프로필 카드가 작성된 것을 확인함.

- <strong>채널 전략 및 PPT 생성</strong>

- `/plan channel` 커맨드를 선택하고 페르소나를 '김지현'으로 지정함.

- "작성된 문서를 PPT 슬라이드나 오피스 독스로 만들어 줘"라고 추가 요청함.

- PPT 스킬이 트리거되어 파이썬 스크립트가 실행됨 (API 또는 라이브러리 실행 가능).

- PPTX 파일로 슬라이드가 자동으로 생성됨.

- <strong>콘텐츠 제작 (카피, 영상 스크립트)</strong>

- 채널별 카피와 영상 스크립트 생성을 요청함.

- 에이전트가 상세한 배경 음악, 썸네일 아이디어 등을 포함한 콘텐츠 스크립트를 만들어줌.

## **N8N과 오픈 코드 통합 및 자동화**

- <strong>통합의 필요성 및 구조</strong>

- 이미지 제작은 스케줄링과 완전 자동화가 더 필요하며, 이를 위해 <strong>워크플로 툴인 N8N과 오픈 코드를 연결</strong>할 수 있음을 강조함.

- 커스텀 노드 개발을 통해 N8N에서 오픈 코드를 연결하여 이미지 및 동영상 생성을 자동화할 수 있음을 보여줄 예정임.

- 오픈 코드 LLM 서버와 N8N 워크플로를 로컬 환경(셀프 호스팅)에 모두 호스팅함.

- 오픈 코드가 API 서버 역할을 하고, N8N 워크플로가 클라이언트가 되어 API 서버를 호출, 에이전트를 가동하는 구조임.

- 이 구조를 통해 N8N의 각 노드가 오픈 코드의 에이전트 또는 스킬과 매핑되어 순차적 처리를 제어할 수 있으며, N8N에서 LLM 서비스들을 무료로 활용할 수 있는 장점이 있음.

- <strong>오픈 코드 서버 실행</strong>

- 터미널에서 `open-code serve --port 4096` 명령어를 입력하여 오픈 코드 서버를 4096 포트로 실행했음.

- <strong>N8N 서버 실행 및 커스텀 노드 설치</strong>

- N8N을 로컬에 설치하고 `n8n start` 명령어로 서버를 실행함.

- N8N 워크플로에서 세팅으로 이동하여 'Community Nodes'에서 `n8n-nodes-opencode-ai` 커스텀 노드를 설치함.

- <strong>N8N과 오픈 코드 연결 설정</strong>

- N8N 워크플로에서 커스텀 노드 'Open-Code'를 선택하고 'Send Message' 액션을 선택함.

- 'Create New Credential'을 눌러 오픈 코드 서버의 포트 번호(4096)와 인증 정보를 입력하여 저장함.

- 기존 오픈 코드 세션 목록 중 '단테 쇼츠 스크립트 생성' 세션을 선택하여 컨텍스트를 이어가도록 설정함.

- 모델은 'Gemini 3 Flash Antigravity'를 선택함.

- 레퍼런스 파일로 이전에 생성한 '인스타그램 카피' 파일을 선택함.

- "카피 내용을 단순하고 명료하게 개선해 줘"라고 메시지를 요청함.

- 'Execute' 버튼을 눌러 메시지를 해당 세션으로 보내 처리하고 응답을 N8N으로 가져옴.

- N8N을 셀프 호스팅하면 파일 시스템 제어가 가능하여 더욱 확장된 활용이 가능함을 강조함.

- <strong>이미지 생성 (Kai.AI 연동)</strong>

- N8N에서 'Execute Command' 액션을 선택하고 'Creative Production' 플러그인의 'Create Image' 커맨드를 선택함.

- 이미지 생성 서비스인 <strong>Kai.AI (kai.ai)</strong>를 이용함.

- Kai.AI에 가입하고 API 키를 생성함.

- 터미널에서 'Auth Manager' 스킬을 사용하여 Kai.AI의 API 키를 등록함 (환경 변수 파일에 안전하게 관리됨).

- 외부 서비스 인증 등록도 에이전트를 통해 자동화할 수 있음을 보여줌.

- N8N의 커맨드 노드에서 'Create Image' 커맨드를 선택함.

- 레퍼런스 파일로 개선된 '인스타그램 카피' 내용을 선택함.

- 임시 세션(Temporary Session)을 사용하여 일회성으로 작업을 진행함.

- N8N에서 보낸 신호가 오픈 코드 에이전트를 가동하여 이미지를 생성하고 파일 시스템에 저장함.

- 모델을 명시하지 않아도 에이전트가 적절한 모델(GPT 45 이미지)을 선택하여 이미지를 생성했음.

- 생성된 인스타그램용 이미지를 확인했음.

- <strong>동영상 생성 (Kai.AI 연동 및 쇼츠 자동화)</strong>

- N8N에서 다시 'Execute Command' 액션을 선택하고 'Create Video' 커맨드를 선택함.

- 임시 세션 또는 기존 세션 유지 선택 가능하며, 여기서는 새로운 세션으로 진행함.

- 레퍼런스 파일로 카피 내용을 입력함.

- 시작 이미지로 이전에 생성한 인스타그램 이미지를 선택하고 5초 분량의 영상을 생성하도록 요청함.

- 에이전트는 시지푸스로, 모델은 Kai.AI의 'Grok Imagen' 모델을 선택함.

- 영상 생성에는 시간이 걸릴 수 있어 타임아웃을 길게 설정함.

- API 비용이 발생하지만, 5초짜리 짧은 영상이 생성됨을 확인함.

- 더 긴 영상, 내레이션, 음악, 효과 등도 스킬로 정의하면 충분히 가능함을 언급함.

## **에이전트 기반 비즈니스 워크플로우의 미래**

- N8N에서 다양한 워크플로를 절차적으로 연결하고, 오픈 코드를 엔진으로 사용하면 시지푸스 에이전트의 모든 기능을 활용하여 에이전틱하게 처리할 수 있음을 설명함.

- N8N의 하나의 노드가 하나의 에이전트 모듈이 될 수 있음을 강조함.

- 블루프린트 예시로 데이터 분석가가 쿼리, 데이터 처리, 문서 생성 및 전송 과정을 에이전트가 처리하는 AX(Agent Experience)로 대체할 수 있음을 제시함.

- 비즈니스 업무를 EDA(탐색적 데이터 분석)처럼 단계적인 레이어로 나누고, 각 레이어 안에 작은 모듈들을 에이전트로 만들면 모듈화 및 시스템화된 목표 달성이 가능함.

- N8N과 오픈 코드의 조합으로 이러한 에이전트 기반의 비즈니스 업무 모듈화를 구현할 수 있다고 생각하며, 실제 비즈니스 업무를 에이전트로 모듈화하여 처리 중임을 밝힘.

- 이는 앞으로 비즈니스의 미래가 될 것이라는 생각을 피력함.

- '단테 커피'라는 가상의 브랜드로 오픈 코드와 N8N 통합을 통한 전체 에이전트 비즈니스 워크플로 구축 방법을 알아봤음.

- 1인 마케팅 에이전시의 가능성을 살펴보았으며, 2026년 캐즘이 깨지는 시기에는 <strong>개인화 에이전트가 선택이 아닌 필수</strong>가 됨을 재차 강조함.

## **커뮤니티 및 도서 이벤트 안내**

- <strong>단테랩스 지식 커뮤니티</strong>

- 카카오 오픈 채팅방으로 오픈함.

- AI 에이전트와 자동화에 관한 열린 지식의 장이며, 매일 최신 AI 뉴스를 가장 빠르게 전달함.

- 입장 링크는 영상 설명란과 고정 댓글에 있음을 안내함.

- <strong>N8N 업무 자동화 도서 제목 공모 이벤트</strong>

- 비개발자를 대상으로 한 N8N 업무 자동화 도서를 상반기 출간 목표로 집필 중임.

- 참여자 중 총 50명을 추첨하여 1만 원 상당의 기프티콘을 증정함.

- <strong>당첨자 추첨 방식</strong>: 국제 공인 기프티쇼 비즈의 자동 추첨 시스템을 통해 인위적인 개입이나 중복 참여 없이 아주 투명하게 진행됨.

- <strong>설문 참여 방법 (1분 소요)</strong>

- 단테랩스 오픈 채팅방 입장.

- 채팅방 상단 공지 사항의 고정된 설문 링크 클릭.

- N8N 도서 제목 작성 후 제출.

- 당첨자에게는 1월 5주차에 문자로 경품을 즉시 발송할 예정임을 알림.
