## **GPT 5.4 공개 및 주요 기능**

- 전문 작업을 위해 설계된 모델임.

- ChatGPT API, Codex 등 다양한 서비스에 GPT 5.4가 적용됨.

- 업계 최고 수준의 역량을 갖춘 GPT 5.3 Codex를 기반으로 모델 능력이 전반적인 업무에서 향상됨.

- 각종 도구 및 소프트웨어 환경에서 성능 개선됨.

## **벤치마크 및 성능 향상**

- 전반적으로 GPT 5.3 Codex에 비해서도 성능이 크게 향상됨.

- 이전 GPT 5.2와 비교하여 점수들이 상당히 높아짐.

- <strong>브라우저 조작 및 UI 조작 능력:</strong> "브라우저 컴프" 점수가 매우 많이 높아짐.

- <strong>전문 지식 기반 업무:</strong> 에이전트가 44개 직종에서 명확히 정의된 지식 업무 GDP 발에서 83%를 기록함.

- 업계 전문가 기준선을 훨씬 뛰어넘어 경제적 가치가 있는 작업을 수행할 수 있음.

- 스프레드시트, 문서, 프레젠테이션 품질이 훨씬 좋아진 것으로 판단됨.

- <strong>컴퓨터 사용 및 비전 기능 (OSWorld Verified):</strong> AI 에이전트 기능 강화와 함께 컴퓨터 조작 및 비전 기반 작업 수행 능력이 매우 많이 올라감.

- <strong>물리 추론 태스크 (Artificial Analysis):</strong> "리서치 레벨 피직 리즈닝"에서 이전 Gemini 3.1 등 대비 엄청난 점수 향상을 보여줌.

- GPT 5.4 Pro X하이 기준으로 30%까지 점수가 향상됨.

- 2025년 11월 기준 불과 몇 개월 만에 최고 점수가 9%에서 30%로 급증함.

- <strong>전체 종합 점수:</strong> Artificial Analysis 기준 아직은 이전 모델들과 동점이나, 몇 주 후 순위 변동이 예상됨.

- GPT 5.4가 새로운 SOTA(State-of-the-Art) 모델 중 하나로 등극함.

- <strong>이미지 기반 웹사이트 제작:</strong> 이미지 기반으로 웹사이트를 만들 수 있음. (시각적 처리 능력 향상 덕분)

- <strong>원샷 프롬프트 기반 3D 모델링/시뮬레이션:</strong> 프롬프트 한 번으로 다양한 시뮬레이션 제작 가능함.

- 플라이트 시뮬레이터 3D 모델링이 매우 잘 구현됨. (금문교 다리, 자동차 등)

- 테마파크 시뮬레이션 (롤러코스터 타이쿤처럼 관객 배치까지) 제작 가능함.

- RPG 게임도 원샷으로 제작 가능함.

- <strong>컴퓨터 조작 업무:</strong> 브라우저 UI 스크린샷을 보고 인터페이스 클릭, 이메일 전송, 캘린더 일정 관리 등 컴퓨터를 조작하는 업무에 능숙함.

- 메일이 오면 마우스 클릭으로 답변을 작성하고 라벨을 만드는 등 작업 수행함.

- 구글 캘린더 초대 보내는 것도 UI를 조작하여 수행함.

- <strong>마인크래프트 클론 제작:</strong> VoxelsCraft를 통해 마인크래프트 클론을 딸깍으로 간단하게 만들 수 있음.

- 실제로 HTML 게임으로 제작된 클론을 실행해 보니 잔디, 흙, 돌 블록 파괴 등 기능이 잘 구현됨.

- 떨어지면 초기화되는 기능도 구현됨.

- <strong>GPT 4 대비 퀄리티 향상:</strong> 2024년 GPT 4로 제작했던 결과물과 동일한 프롬프트로 5.4를 사용하니 퀄리티 차이가 현저하게 나타남.

- <strong>After Effects 연동:</strong> GPT 5.4를 애프터 이펙트(After Effects)의 Atom 익스텐션에 사용하여 SVG 애니메이션 작업을 시킬 수 있음.

- 자전거 타는 펠리컨 챌린지 같은 작업을 통해 정교한 애니메이션 구현이 가능함.

- Gemini 3.1보다 더 디테일하게 잘 구현되는 것을 확인함.

## **GPT 5.3 Instant 모델 업데이트**

- ChatGPT에서 가장 많이 사용되는 모델로, 빠른 응답 속도가 특징임.

- 성능이 더 좋아짐.

- <strong>거절 판단 개선:</strong> 책임 회피하는 표현을 줄여 일반적인 채팅에서도 성능이 좋아진 것으로 판단됨.

- 기존 버전은 6월 3일부로 지원이 종료될 예정임.

- 사실 정확도와 창의적 글쓰기 품질도 강화됨.

- <strong>테스트 결과:</strong> "세차장이 50m 떨어져 있다면 걸어갈까 운전할까?" 질문에 "무조건 걸어가는 게 낫다. 차 움직이면 오히려 손해다"라고 답변함.

- 이후 "걸어왔는데 세차는 어떻게 하냐"는 질문에 "제가 실수했네요"라고 인정하는 모습을 보임.

- 여전히 아쉬운 점이 일부 존재함.

## **Claude 업데이트 소식**

- <strong>트위치 클로드 (Switch to Claude) 기능:</strong> 다른 AI 서비스 사용자들이 선호도와 컨텍스트 정보를 클로드로 그대로 가져올 수 있는 기능이 공개됨.

- 한 번의 복사 붙여넣기로 클로드의 메모리를 업데이트하고 사용자가 중단했던 부분부터 다시 시작할 수 있음.

- OpenAI의 논란으로 넘어오는 사용자들을 위해 전환 기능을 마련한 것으로 판단됨.

- "Import memory to Claude"라는 기능으로, 대화 내역을 가져올 수 있도록 가이드를 제공함.

- <strong>데스크톱 버전 로컬 예약 작업 기능:</strong> Claude Code 데스크톱 버전에 스케줄 태스크 (Scheduled Task) 기능이 출시됨.

- OpenFlow 등 다른 플랫폼의 예약 작업 기능과 유사함.

- 매일 오전 9시마다 데일리 코드 리뷰를 진행하거나 특정 시간, 요일 등으로 설정하여 주기적인 작업을 자동 실행할 수 있음.

- "loop" 슬래시 명령어를 사용하여 5분 동안 배포 상태를 확인하고 알림을 주는 등 반복적인 작업을 시킬 수 있음.

- <strong>보이스 모드:</strong> Claude Code 안에서도 보이스 모드가 지원됨.

- "보이스"라고 입력하면 리스닝 모드가 활성화되어 음성으로 코딩 작업을 수행할 수 있음.

- <strong>무료 온라인 강의:</strong> Anthropic에서 클로드 코드 등 자사 모델 사용법을 알려주는 공식 무료 온라인 강의 "Anthropic Courses"를 공개함.

- <strong>Gemini 3.1 Flash Lite 모델 공개:</strong> 저렴하면서도 성능이 좋게 출시됨.

- 가볍게 만들 수 있는 LLM 포함 프로덕트 제작에 좋은 옵션이 될 것으로 보임.

- 벤치마크 결과 전반적으로 성능이 향상됨.

## **LLM 상호작용 방식 변화: CLI 기반 접근의 부상**

- <strong>CLI (Command Line Interface) 기반 접근의 실용성 부각:</strong> MCP (Model-Controller-Programmer)보다 CLI 기반 접근이 더 실용적이라는 의견이 나옴.

- LLM이 이미 명령줄 도구 사용에 능숙함.

- Claude Code 등에서 CLI 명령어를 스스로 입력하며 작업을 수행하는 것을 보임.

- 굳이 MCP를 통해 서비스를 연결하기보다 CLI 기반 도구들을 연결하여 LLM이 직접 명령어를 날리게 하는 것이 더 효율적이라는 판단임.

- CLI는 일반적으로 인간과 LLM이 동일한 환경에서 실행 및 디버깅 가능하여 더 단순함.

- MCP보다 CLI가 더 단순하고 신뢰성 높은 선택지라는 의견이 나옴.

- 사람이 사용하던 API나 CLI를 AI도 성능 향상으로 잘 사용할 수 있게 됨.

- <strong>Google Workspace CLI:</strong> 구글 워크스페이스(구글 드라이브, Gmail, 캘린더, Docs 등)를 CLI 터미널에서 조작할 수 있도록 하는 GWS (Google Workspace CLI)가 화제가 됨.

- AI 에이전트용 스킬 100개 이상을 포함하여 LLM과 CLI 인터페이스로 통합하여 사용할 수 있음.

- AI 에이전트가 구글 드라이브에 파일 저장, 이메일 전송 등 구글 제품과 연동된 작업을 스스로 수행하도록 할 수 있는 좋은 수단이 됨.

## **영상 생성 AI 모델 동향**

- <strong>LTX 2.3 버전 출시:</strong> 오픈 소스로 공개된 영상 생성 AI 모델인 LTX의 퀄리티가 더욱 좋아짐.

- 상당히 괜찮은 영상 퀄리티를 오픈 소스 모델로 생성할 수 있음.

- 기본적으로 세로형 영상도 바로 만들어낼 수 있으며, 시작 및 종료 프레임 설정 등 고급 기능도 가능함.

- 오픈 소스로 내 컴퓨터에서 인터넷 없이도 영상 생성이 가능함.

- VM 용량이 약 46GB 정도로 무거운 편이나, 양자화 모델 등이 나올 경우 온디바이스에서도 구동 가능할 것으로 예상됨.

- Comfy Cloud를 통해 무료로도 체험 가능하며, RTX 6000 Pro GPU 96GB VM이 제공됨.

- 5초 영상 생성에 RTX 6000 기준으로 1분 6초가 소요됨.

- 연매출 1천만 달러 이하까지는 상업용으로 무료 사용 가능함.

- <strong>Helios: 실시간 롱 비디오 제너레이션 모델:</strong> 14B 모델로 실시간 영상 생성이 가능함.

- 하나의 HGPU에서 19.5 FPS로 실시간 영상 출력이 가능함.

- 일반적인 하이 퀄리티 영상 생성 정도는 아니지만, 실시간이라는 점을 고려하면 나쁘지 않은 퀄리티임.

- 실시간 무한 생성으로 영상을 뽑아낼 수 있음.

- 이전 실시간 생성 모델들보다 퀄리티가 많이 좋아짐.

## **소형 LLM 및 관련 이슈**

- <strong>Qwen 3.5 Small 모델 시리즈 출시:</strong> 이전의 미디엄 버전에 이어 0.8B, 2B, 4B, 9B 같은 아주 작은 모델들이 출시됨.

- 작은 모델임에도 불구하고 꽤 좋은 성능을 보여줌.

- 휴대폰 등 로컬 디바이스에서도 AI 모델을 사용해 볼 수 있게 됨.

- <strong>Qwen 개발팀 핵심 연구진 대거 사임:</strong> Alibaba 모델인 Qwen 팀의 핵심 연구진이 대거 사임함.

- 최근 공개된 Qwen 3.1 모델 시리즈의 향후 개발이 불투명해짐.

- 팀 리더를 포함한 여러 핵심 인력들이 동반 퇴사함.

- 이는 중국발 오픈소스 AI 연구 생태계의 중요한 변곡점이 될 수 있음.

## **하드웨어 최적화 LLM 모델 선택 도구**

- <strong>LMFit (LM Find It) 출시:</strong> 현재 사용하는 하드웨어에 어떤 LLM 모델을 사용해야 할지 알려주는 도구임.

- 단 한 명령어("lm fit")로 하드웨어(RAM, CPU, GPU)에 맞춰 LLM 모델 크기를 최적화함.

- 하드웨어를 감지하고 품질, 속도, 정확성, 컨텍스트 측면에서 각 모델의 점수를 매김.

- 시스템에서 실제로 잘 동작하는 모델을 "Good", "Perfect" 등으로 표시하여 추천함.

## **메타 스마트 안경 개인정보 유출 논란**

- <strong>"우리는 모든 것을 본다" 논란:</strong> 메타 스마트 안경(Ray-Ban Stories) 개발 노동자들이 개인 정보 유출 논란을 제기함.

- 안경에 달린 카메라로 녹화된 영상 데이터가 메타로 전송되어 케냐 하청 업체 노동자들이 직접 검수 및 주석 처리한 것으로 드러남.

- 나체, 성행위, 금융 정보 등 사생활 영상 및 음성 데이터까지 사람이 직접 분류하는 것으로 확인됨.

- 메타의 AI 이용 약관에는 사용자의 대화 내용이 자동 또는 인간 검토의 대상이 될 수 있다고 명시되어 있으나, 데이터 처리 및 저장 범위는 명확히 공개되지 않음.

- 사용자는 민감한 데이터를 안경을 쓰고 볼 경우 하청 업체 직원들이 볼 수 있으므로 동의 설정을 끌 필요가 있음.
