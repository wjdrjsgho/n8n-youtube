## **요약할 내용 없음**

- 제공된 내용을 확인해 본 결과, 요약할 텍스트가 존재하지 않음.

- 요약을 진행하려면 구체적인 텍스트 내용을 제공해야 함.

- 텍스트가 제공되면 요청 조건에 맞춰 상세하게 요약할 예정임:

- 전체 내용을 순서대로 구조화할 예정임.

- 주요 주제마다 <h3> 태그로 소제목을 작성하고 <strong> 태그로 감싸 굵게 처리할 예정임.

- 각 소제목 아래는 순서 없는 리스트()와 리스트 항목(
- ) 형태로 bullet point를 사용하며 내용을 풀어나갈 예정임.

- 계층 구조가 필요한 경우 중첩 리스트( 안에 )를 사용하여 하위 항목을 표현할 예정임.

- 각 항목은 간결하면서도 구체적으로 작성하며, 예시나 논리 흐름이 빠지지 않게 포함할 예정임.

- 내용이 많으면 길게 작성하더라도 지나친 축약은 하지 않을 예정임.

- 감탄사나 미사여구는 배제하고 정보 중심으로 작성할 예정임.

- 모든 문장은 반드시 음슴체로 작성할 예정임.

- 한글로 작성할 예정임.

- 화자가 1인칭 관점으로 말하듯이 하되, '나는', '저는' 같은 표현은 절대 사용하지 않을 예정임.

- 화자의 서술을 부연 설명하는 표현은 절대 생략하지 않을 예정임.

- 마지막에 해시태그 목록을 작성할 예정임.

- <html>, <head>, <body>, <!DOCTYPE> 태그는 절대 포함하지 않고, <h3> 태그부터 바로 시작할 예정임.

- <style> 태그나 인라인 CSS(style="...")는 절대 사용하지 않을 예정임.

- 굵은 글씨는 <strong> 태그를 사용할 예정임.

- 마크다운 코드 블록(```)을 사용하지 않고 순수 HTML 텍스트만 반환할 예정임.
