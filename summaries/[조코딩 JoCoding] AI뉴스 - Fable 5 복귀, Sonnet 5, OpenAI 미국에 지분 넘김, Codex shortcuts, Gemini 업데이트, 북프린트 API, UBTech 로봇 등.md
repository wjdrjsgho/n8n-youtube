## **페이블(Fable) 재개 및 이용 조건**

- 페이블이 6월 31일 해제되어 7월 1일부터 사용할 수 있게 됨.

- 안타깝게도 사용 기간에 제한이 있음.

- 구독자의 요금 안에서 쓰려면 7월 7일까지 주간 사용량 한도의 최대 50%까지만 쓸 수 있음.

- 7월 7일 이후부터는 크레딧을 통해 이용할 수 있어 가격이 확 비싸짐.

- 현재는 클로드 맥스 200달러 플랜 결제 시 오퍼스 기준으로 약 8,000달러 가치를 하는 등 어마어마하게 저렴하게 쓸 수 있는 기회임.

- 구체적인 가격은 아우프 토큰이 제일 직관적인데, 50달러당 100만 토큰으로 오퍼스 가격의 두 배임. 비싸서 못 쓰는 모델이 될 수도 있을 듯함.

- 아르티피셜 아나리시스 조사 기관에 따르면 코스트 인텔리전스가 너무 비쌈. GPT 5.5에 비해서도 두 배가 넘음.

- 7월 7일 전까지 특가로 페이블을 이용할 수 있는 기간이므로 팍팍 써야 함.

- 약간 안타까운 점은 코딩 및 디버깅 업무에 페이블을 쓰다가 갑자기 오퍼스로 다운그레이드되는 경우가 있다고 함.

- 정상적인 요청을 더 자주 악성으로 표시하는 단점이 있다고 소개됨.

- 위험하다고 잘못 판단하여 오퍼스를 사용하는 경우가 종종 발생하는 것임.

- 아레나에서 측정한 결과, 거의 성능이 유지된 채 안전 필터만 조금 더 걸어 출시된 상황으로 보임.

## **페이블 활용 팁 및 전략**

- 한정 기간 동안 해야 할 일이 있음.

- 페이블 5에게 오퍼스 4.8용 스킬을 작성하게 해두자는 이야기가 화제가 됨.

- 최강 모델에게 평소 사용하는 모델의 생각 방식을 가르치게 하는 것임.

- 쓸 수 있을 때 어려운 문제는 빨리 시켜두는 것이 좋음.

- 페이블 5 토큰을 절약하는 꿀팁이 나옴.

- 클로드에서 이미지 토큰 비용 산정이 애매하게 되어 이미지 안의 텍스트 양은 보지 않는다고 함.

- 이미지 한 장을 넣으면 그 안에 어떤 내용이 들었든 토큰을 똑같이 먹는다는 것임.

- 이미지 안에 텍스트를 빽빽하게 적어 넣으면 이미지 하나의 토큰만 먹음.

- 이 방법을 사용하면 60% 절감 가능하며, 1900 토큰을 약 59%에서 70%까지 낮춰 준다고 함. 해치될지 모르나 당분간은 유효할 듯함.

- 엔트로픽 관계자가 페이블 활용 방법에 대한 팁을 남김.

- 페이블이 너무 자세하게 프롬프트를 쓰면 오히려 안 좋다고 함.

- 어떤 경로를 찾아가는 길에 너무 빡세게 하네스를 걸면 고정된 방식으로 이동하여 창의적인 탐색을 할 수 없다는 것임.

- 그렇다고 너무 열어놔도 잘 안 되므로 적정선을 지키라는 팁임.

- 요약본을 보면, 브레인스토밍, 프로토타입, 인터뷰를 먼저 진행해 아이디어를 잡으라는 것임.

- 어떤 것을 구현해야 하면 가능한 방법들을 나열해 먼저 브레인스토밍 프로토타이핑 인터뷰를 진행해야 함.

- 이것으로 저렴한 비용으로 방법을 탐색할 수 있음.

- 방법을 정한 다음 진행해야 큰 수정 비용을 줄이는 최고의 방법이라고 강조함.

## **클로드 소넷 5 공개 및 평가**

- 엔트로픽에서 클로드 소넷 5를 공개함.

- 기존 오퍼스, 소넷, 하이쿠 세 가지 모델 중 중간 모델인 소넷의 다음 버전임.

- 결론적으로 딱히 쓸 이유가 없어 보임.

- 소개에 따르면 괜찮게 나오긴 했으나 오퍼스보다는 안 좋고 애매하게 좋은 성능임.

- 저렴하면 쓸 만하겠지만, 가격을 보면 나쁘지 않은 정도임. 할인 기간 동안 100만 토큰당 15달러로 중간 모델 정도 가격임.

- 하지만 작업당 토큰 비용을 측정해 보니 해결된 문제당 비용은 더 비싸졌다고 함.

- 이전 소넷 버전보다 비용이 올라갔고, 더 놀라운 것은 이전 오퍼스보다도 비쌈.

- GPT 5.5는 훨씬 저렴한데, 소넷 5를 쓸 이유를 이해하기 어려운 가격임.

- 심지어 전체 벤치마크를 돌리는데 페이블보다 비쌌다고 함.

## **북프린트 API 소개**

- 북프린트 API는 개발자들도 빠르게 실물 책을 출력하고 배송까지 보낼 수 있는 API임.

- 실물 책 제작 및 배송은 원래 굉장히 어려운 과정인데, API 하나로 간편하게 처리할 수 있음.

- 실제로 '코딩 친구'라는 책을 뽑아 봤는데, 실물 책이 제작되는 것을 볼 수 있음.

- API 호출 한 번으로 책을 만들 수 있음.

- 이 기술 스택으로 동화책 SaaS 서비스를 만들어 책 제작부터 배송까지 풀 튜토리얼을 올려놓았음.

- 동화책 서비스, 커플 앨범, 결혼식 앨범 등 책 형태로 나오면 좋을 만한 SaaS 서비스를 만들 수 있음.

- 이 API를 만든 곳은 스위트북으로, 달력, 퍼즐 등을 전문으로 만들던 회사임.

- 개발 문서도 매우 잘 되어 있어, 링크를 던져주거나 AI 에이전트에게 LLM.txt를 던져주면 문서를 읽어 서비스에 붙일 수 있음.

- 개발자 입장에서도 복잡한 인쇄 공정 생각 없이 PDF만 만들어서 API로 보내면 책을 바로 만들어 볼 수 있음.

- AI 동화책뿐 아니라 여행 포토북, 일기책, 교육 결과물, 크리에이터 굿즈 등 다양하게 활용 가능함.

## **메타 관련 논란 및 기술 발전**

- 인스타그램 릴스에 노출된 메타 글래스 광고가 논란이 됨.

- 사용자 본인의 프로필 사진을 끌어와 그 사람의 남자친구를 겨냥하는 초개인화 광고가 뜨면서 논란이 확산됨.

- 기존 개인화 광고는 알고리즘, 연령대, 사는 곳 등을 기반으로 타겟팅했으나, 이제는 프로필 사진까지 활용하는 것임.

- 내가 업로드한 이미지나 친구 관계를 이용해 광고를 하는 것이므로 논란이 될 수밖에 없음.

- 마이크 도청 불안과 결합된 거부감이 확산되고 있음. 해당 사례가 조작일 수도 있다는 이야기도 있으나 화제가 됨.

- AI 시대가 될수록 수집된 데이터를 활용할 방법이 많아져 개인의 맥락을 AI로 추론해 정교한 타겟팅 광고가 가능해질 것이며, 더 강력해질 수 있음을 우려함.

- 메타가 비침습 뇌 텍스트 디코더를 공개함.

- 뇌를 분석해 생각만으로 텍스트를 추측하는 기술임. 생각만 하면 텍스트가 입력되는 것.

- 머리에 쓰는 디바이스만 꽂고 있으면 생각만으로 글자를 쳐서 바이브 코딩도 시켜 볼 수 있을 것임.

- 참여자 전체 평균 단어 정확도는 61%, 최고 성능 참가자는 78%의 단어 정확도를 보임.

- 메타의 클라우드 사업 진출 검토 소식이 있음.

- 메타는 AI 데이터 센터에 막대한 돈을 투자함. 데이터 센터 운영 규모를 보면 아마존, 마이크로소프트, 구글 같은 클라우드 사업자와 유사한 거대한 규모임.

- 지금까지는 자체 AI 데이터 학습 등에 사용되던 컴퓨팅 파워였으나, 메타도 클라우드 사업 진출을 검토 중이라는 소식이 화제가 됨.

- 남는 GPU 연산 자원을 외부 기업에 판매하는 클라우드 사업 진출 검토 보도에 주식 시장이 출렁거림.

- GPU가 남아도는 것이 아닌가 하는 생각에 지금까지의 메모리, GPU 투자 열풍이 둔화될 수 있다는 우려로 폭락 사태가 있었음.

- 개인적인 생각으로는 메타가 과대 투자를 해서 남는 것이지, 다른 곳은 아직 부족한 상황일 수 있다고 함.

- 메타도 클라우드 사업에 진출하면 다른 회사들이 메타의 GPU를 빌려 쓸 수도 있어 메타가 꽤 강력해질 수 있을 것으로 봄.

- 중국의 배달 업체 메이안(Meituan)이 롱켓 2.0 모델을 출시함.

- 모델 자체 성능도 괜찮으나, 엔비디아 GPU 없이 만들어졌다는 점이 놀라움.

- 심지어 오픈 웨이트 모델임.

- 세계 최초로 중국산 AI 칩만을 사용해 학습과 추론을 모두 수행한 1조 파라미터 AI 모델임.

- 중국산 AI 칩 5만 개 규모 클러스터가 사용됨. 이제 엔비디아 없이도 이런 엄청난 모델이 나올 수 있게 됨.

- 벤치마크를 보면 터미널 벤치 등에서 제미나이보다 한참 앞서는 등 꽤 잘 나왔음. 이런 성능의 모델이 오직 중국 AI 칩만으로 학습되었으니 긍정적임.

- MR플로우 오픈소스 이미지 모델 가속화 방법이 공개됨.

- 플럭스나 Z 이미지, 펜 등 오픈소스 이미지 모델에서 학습 없는 가속화 방법임.

- 이 방법을 적용하니 굉장히 빠르면서도 성능이 높은 방법이 됨. 기존 모델에 적용할 수 있음.

- 속도 향상: 플럭스데 8.25배, 퀸 이미지 10배, Z 이미지 터보 21배, 퀸 이미지와 다른 것을 결합했을 때는 25배까지 이미지 생성이 빨라짐. 순식간에 이미지를 찍어낼 수 있음.

## **유비테크 U1 로봇 공개 및 성공**

- 유비테크(UBTECH)라는 로봇 업체가 U1 시리즈 티저 영상에 이어 실제 로봇을 공개함.

- 팔을 눌러보면 실제로 꼬집을 수 있을 정도로 구현됨. 다양한 의상과 컨셉으로 출시됨.

- 성과가 놀라움. 공개 직후 누적 예약만 약 1만~1만 3천 대를 돌파함. 시장의 관심과 폭발적인 인기를 끌고 있음.

- 특징: 매우 유사한 외형과 표정, 눈동자 움직임, AI 기반 감성 대화, 사용자 맞춤 상호 작용, IP 캐릭터 외형 커스터마이징 지원.

- 1인 가구, 고령층의 정서적 동반자 수요를 주요 타겟으로 설계됨.

- 가격은 2천만원에서 3천만원 정도이며, 최고 사양은 2억 원 이상이라고 함. 출시될 때 팔릴까 싶었는데 시장 수요로 증명됨.
