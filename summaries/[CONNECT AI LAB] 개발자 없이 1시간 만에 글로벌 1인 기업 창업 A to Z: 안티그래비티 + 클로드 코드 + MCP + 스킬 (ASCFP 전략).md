## **AI를 활용한 서비스 개발의 어려움 및 해결 전략**

- AI를 활용한 코딩이 쉽다고들 하나, 실제 개발 시 다음과 같은 문제에 직면하는 경우가 많아 프로젝트를 쉽게 포기하게 됨.

- 토큰 한도 초과 및 추가 결제 문제 발생함.

- 결제 페이지, 로그인, 회원가입 데이터 연동 등 복잡한 작업이 수반됨.

- 전체 과정을 한 번에 해결할 수 있는 방법으로 ASCFP 전략을 제시함.

- ASCFP 전략은 Anti-Gravity, Claude Code, MCP (Firebase, Pay)를 연결하여 글로벌 결제까지 가능한 웹사이트를 한 번에 개발하는 방식임.

## **화자의 목표 및 접근 방식**

- 세계 최고의 인공지능 교육을 제공하려는 목표를 가짐.

- 글로벌 확장을 위해 개발 및 콘텐츠 제작을 진행하던 중 잠시 중단함.

- 에듀테크 1인 기업 운영 연구를 통해 현재 가능한 일들이 많음을 발견함.

- 우선 한국 사용자들에게 좋은 교육을 제공하고, 안정화 이후 글로벌 확장 예정임.

- 올해(당해)까지는 한국 시장에 집중할 계획임을 밝힘.

## **Anti-Gravity 소개 및 활용**

- Anti-Gravity는 에이전트 기반의 노코드/로우코드 바이브 코딩 툴임.

- 사용자의 지시만으로 여러 에이전트가 동시에 작동하여 웹사이트, 콘텐츠, 앱 등을 자동으로 생성함.

- 스킬, 외부 도구(MCP), 타사 AI 도구(Claude Code) 등을 연결하여 에이전트의 지능을 높일 수 있음.

- 처음에는 어렵게 느껴질 수 있으나, 반복 학습을 통해 쉽게 활용 가능하며, 실제 60~70대 구독자들도 웹사이트 제작 및 자동화를 수행 중임을 언급함.

- 사용자들이 쉽게 따라 할 수 있도록 Anti-Gravity 설치만으로 바로 시작할 수 있게 교육 과정을 구성함.

## **학습 방법론**

- 다양한 분야의 개발 과정을 지속적으로 공유하며, 평소에 가볍게 따라 해 볼 것을 권장함.

- 개인적인 아이디어가 떠올랐을 때, 기존의 유사한 예시를 변형하여 자신의 아이디어에 맞춰 적용하는 방식을 추천함.

## **개발 환경 설정 (Setup)**

- <strong>Anti-Gravity 설치</strong>

- 공식 웹사이트에서 다운로드 및 설치함.

- 설치 후, 프로젝트 폴더를 생성하고 열기 버튼을 통해 작업 공간을 설정함.

- <strong>Claude Code 설치 및 활용</strong>

- VS Code의 확장 프로그램 메뉴에서 'Claude Code'를 검색하여 설치함.

- Anti-Gravity가 기본적으로 Google Gemini AI를 사용하지만, 코딩에는 Claude AI가 강점이 있어 두 AI를 함께 활용하는 전략임.

- 설치 후 터미널을 열어 제공된 코드를 입력하여 Claude Code를 실행함.

- <strong>비용 효율적 AI 활용 전략</strong>

- Anti-Gravity의 멤버십(프로, 울트라)은 사용 한도가 있으며, 울트라 멤버십은 비용이 높음.

- Claude Code 역시 멤버십이 있으나, API 키를 발급받아 연결함으로써 비용을 효율적으로 관리할 수 있음.

- API 키를 발급받아 Claude Code에 연결하여 보조 AI 개발 에이전트로 활용함.

- <strong>MCP (Meta Connect Platform) 설정 및 도구 연동</strong>

- <strong>스킬(Skills)의 중요성</strong>

- 스킬은 AI 에이전트에게 명령을 효율적으로 전달하는 방법임.

- 긴 대화 기록으로 인한 속도 저하 및 토큰 낭비를 방지하고, 에이전트가 특정 단어에 반응하여 필요한 스킬만 발동되도록 설정할 수 있음.

- 개발 시 모든 텍스트 생성이 비용으로 연결되므로 스킬 활용은 매우 중요함.

- <strong>MCP를 통한 도구 연결</strong>

- MCP는 Anti-Gravity와 외부 도구들을 연결해 주는 플랫폼임.

- 웹사이트 개발에 필요한 다양한 도구를 직접 만드는 대신, 외부에서 잘 만들어진 도구를 가져와 사용하는 개념임 (건축에 필요한 포크레인, 망치 등을 구매하거나 빌리는 것에 비유함).

- 주요 연결 도구는 Firebase와 PayPal임.

- <strong>Firebase 설치</strong>

- MCP 서버에서 Firebase 도구를 찾아 설치함 (데이터베이스, 인증 등에 사용됨).

- <strong>PayPal 설치 및 설정</strong>

- MCP 서버에서 PayPal 도구를 찾아 설치함.

- PayPal 계정 생성 후, 샌드박스(테스트 환경) 또는 라이브(실제 환경) 앱을 생성하여 클라이언트 ID와 시크릿 ID를 발급받음.

- 발급받은 ID를 MCP에 입력하고, 액세스 토큰 및 환경(샌드박스) 값을 설정하여 연동을 완료함.

## **웹사이트 개발 과정**

- <strong>스킬 활용을 통한 개발 시작</strong>

- 준비된 스킬 코드를 Anti-Gravity 작업 공간에 복사하여 붙여 넣은 후 전송함.

- 이 과정에서 에이전트가 폴더, 페이지, 코딩 작업을 자동으로 수행함.

- 작업 과정을 한국어로 상세하게 설명해달라고 지시할 수 있음.

- 에이전트가 'Agent', 'Skills', 'Money Website' 등의 폴더를 생성하고, 'skills.md'라는 작업 매뉴얼 문서를 작성함.

- <strong>에이전트의 계획 수립</strong>

- Anti-Gravity의 모드를 'Fast'에서 'Planning'으로 변경하고, 'Gemini 3 Pro' 모델을 선택함.

- 메타데이터를 통해 '수익화' 관련 단어가 스킬을 발동시키도록 설정함.

- '수익화 웹사이트 바로 개발하자'고 지시하면, 에이전트가 'skills.md'를 분석하여 웹 AI 수익화 아이디어를 제안함 (예: AI 자기소개 생성기, 이력서 기반 맞춤형 커버레터 생성).

- 제안된 아이디어 중 하나를 선택하면, 에이전트가 웹사이트 개발을 위한 구체적인 작업(Task) 목록을 상세하게 계획함.

- <strong>초기 웹사이트 결과 및 기능 설명</strong>

- 'Proceed' 버튼을 눌러 개발을 시작하면, 에이전트가 웹사이트를 자동으로 생성함.

- 초기 생성된 웹사이트는 기본적인 디자인을 가짐.

- 웹사이트의 주요 기능은 다음과 같음:

- 랜딩 페이지 제공함.

- Google 로그인 (Firebase 인증 연동)을 통해 복잡한 가입 절차 없이 원클릭 로그인을 지원함.

- PayPal 결제를 통한 수익화 기능을 포함함.

- <strong>Claude Code를 이용한 디자인 개선</strong>

- Anti-Gravity에서 많은 토큰을 사용했을 경우, 세부적인 디자인 개선은 Claude Code를 활용함.

- 터미널에서 Claude Code를 실행한 후, '프로젝트 전체 체크하고 디자인 깔끔하게 변경해 줘. UI/UX 체크해서 글로벌 사용자들이 편하게 디자인해 주고, 눈이 편안한 색으로 해 줘'와 같이 구체적인 지시를 내림.

- 이러한 과정을 통해 웹사이트의 디자인과 사용자 경험(UI/UX)을 개선함.

- <strong>최종 결과 및 기능 시연</strong>

- 개선된 디자인의 웹사이트가 완성됨.

- 사용자가 이력서 정보를 입력하고 커버레터 생성을 요청하면, 결제 페이지(PayPal 또는 신용카드)로 이동하여 결제 후 서비스를 이용할 수 있는 완벽한 기능의 글로벌 웹사이트가 구축됨.

## **결론 및 성과 요약**

- Anti-Gravity, 스킬, Claude Code, MCP, Firebase, PayPal을 연동하여 아이디어 구상부터 웹사이트 개발, 로그인, 결제 페이지, 백엔드 기능까지 한 번에 구현함.

- 개발자나 디자이너 등 여러 명의 인력이 필요했던 작업을 중간 세팅 시간을 포함하여 약 1시간 만에 완료함.

- 초기 세팅(API 등록 등)에 시간이 소요될 수 있으나, 한 번 설정해두면 유사한 글로벌 웹사이트를 다수 만들 수 있는 시스템을 갖추게 됨.

- 한국 시장보다 글로벌 서비스가 유리한 점은 시장 규모가 크고, PayPal 연동이 국내 결제 시스템보다 허가 절차가 용이하다는 점임.

- 사용자들에게 이 과정을 따라 해보고 1인 기업을 바로 시작할 것을 강력히 권유함.
