## **Claude Code 및 Remotion 소개**

- 인트로, 편집, 애니메이션 등 화면상의 모든 것이 Claude Code를 통해 제작되었음.

- 자신을 제외한 모든 요소가 프롬프트로 생성된 것임.

- Claude Code는 외부 앱과 연동이 매우 뛰어난 강력한 AI 코딩 도구이자 비서임.

- Remotion이라는 도구를 Claude Code에 연결하여 모든 그래픽을 생성했음.

- 오늘 몇 분 만에 애니메이션을 만드는 방법을 정확히 보여줄 예정임.

## **Claude Code 설정**

- Claude Code는 데스크톱 앱으로, 컴퓨터에 다운로드해야 함.

- Claude Code를 사용하려면 Claude의 Pro 요금제로 업그레이드해야 함.

- Claude Code 인스턴스를 설정해야 함.

- 새 폴더를 생성해야 하며, 이는 비디오 편집 프로젝트 전용 폴더임.

- 예시로 "animation", "video editing" 등으로 이름을 지정할 수 있음.

## **Remotion 연동 및 기능**

- Claude Code에 Remotion이라는 도구를 사용하도록 프롬프트할 것임.

- Remotion은 React를 사용하여 MP4 비디오를 프로그래밍 방식으로 만들 수 있게 해줌.

- 코드를 통해 이러한 종류의 비디오를 제작할 수 있는 것임.

- Claude Code는 Remotion 라이브러리에 접근하여 다양한 컴포넌트를 활용, 멋진 애니메이션을 조합할 수 있음.

- 비디오 시작 부분에서 보았던 애니메이션과 같은 것임.

- Remotion 웹사이트를 스크롤하며 만들 수 있는 다양한 종류의 콘텐츠를 확인할 수 있음.

- Remotion 웹사이트 상단에 "use with claude code"라고 명시되어 있으며, AI가 제어할 때 매우 원활하게 작동하기 때문에 함께 사용할 것을 권장하고 있음.

## **초기 애니메이션 생성 과정**

- 비디오 시작 부분에 보인 애니메이션을 직접 만드는 과정을 보여줄 예정임.

- Notion 문서에 있는 인트로 스크립트를 복사하여 Claude Code로 가져갈 것임.

- Claude Code에 이 인트로 스크립트를 제공하고, 이 비디오를 위한 애니메이션을 만들도록 지시할 것임.

- Remotion을 사용하여 이 비디오를 만들도록 지시할 것임.

- 그러면 Claude Code가 비디오를 볼 수 있는 링크를 제공할 것임.

- 매우 간단한 프롬프트를 사용할 것임.

- "Remotion을 사용하여 비디오 애니메이션을 만들어 주세요. 다음은 그래픽 비디오를 만들 스크립트입니다."라고 입력한 후 전체 인트로 스크립트를 붙여넣을 것임.

- 궁극적인 목표는 스크립트를 보고, 스크립트의 각 부분을 이해하며, 관련 그래픽을 파악하여 애니메이션을 구축하는 것임.

- 만약 이미 존재하는 비디오를 위해 애니메이션을 만드는 경우, 타임스탬프를 추가할 것임.

- Remotion은 타임스탬프를 이해하며, 비디오의 특정 시점에 무엇을 할지 파악할 수 있기 때문임.

- 예시로, 인트로의 특정 부분에서 Claude Code에 대해 이야기한다면, 해당 타임스탬프에 Claude Code 애니메이션이 나타나도록 할 것임.

- 하지만 현재 말하는 부분을 촬영 및 편집하지 않았기 때문에 정확한 타임스탬프를 알 수 없음.

- 따라서 실제 프로젝트와는 달리 약간 "눈감고" 빌드하는 셈임.

- 인트로는 최대 20~30초 정도 걸릴 것으로 가정하여, Claude Code에 20~30초 분량을 만들도록 지시할 것임.

- 이는 작업할 시간 프레임을 제공하기 위함임.

- 자신의 브랜딩을 추가할 것임.

- YouTube 채널을 위해 흰색 테마에 파란색 악센트, 그리고 현대적인 텍스트 스타일 테마를 사용하도록 Claude Code에 지시할 것임.

- 요청을 보낸 후 Claude Code가 작업을 시작하는 것을 볼 수 있음.

- 많은 팝업이 나타나 명령 실행 및 Remotion 접근을 위한 컴퓨터 권한 승인을 요청할 것임.

- 모든 접근 권한을 허용하면 Claude Code가 모든 작업을 수행할 것임.

## **생성된 애니메이션 검토**

- Claude Code가 작업을 완료했음을 알리는 메시지가 나타남.

- 요청한 모든 요소가 추가되었음을 확인시켜줌.

- "local host link"가 제공됨.

- 실제 비디오 편집 스튜디오는 컴퓨터에 로컬로 호스팅되며, 브라우저를 사용하여 이를 확인하는 방식임.

- 링크를 클릭하면 브라우저에서 로컬 호스팅된 Remotion 스위트 인스턴스를 볼 수 있음.

- 여기서 재생 버튼을 눌러 방금 생성된 애니메이션을 시청할 수 있음.

- 인트로 스크립트를 기반으로 생성된 전체 애니메이션임.

- 멋진 애니메이션, 흰색과 파란색의 브랜드 색상, 스크립트와 관련된 모든 요소가 포함되어 있음.

- 며칠을 기다려야 하는 비디오 편집자 없이도 즉시 이러한 결과물을 얻을 수 있어 매우 놀라움.

- 혼자 작업하며 편집에 시간을 쓰고 싶지 않은 사람에게 매우 유용함.

## **애니메이션 개선 및 반복 작업**

- 개인적으로 애니메이션을 약간 수정하고 싶을 것임.

- Claude Code로 돌아가 수정 사항을 지시할 수 있음.

- 주요 수정 사항은 다음과 같음.

- 모든 콘텐츠를 인트로 화면의 왼쪽으로 중앙 정렬할 것임.

- 그 이유는 화면 오른쪽에 자신의 모습을 추가하고 싶기 때문임.

- 이런 방식으로 오른쪽에 자신이 있고 왼쪽에 애니메이션이 보이게 할 것임.

- 텍스트를 줄이고 애니메이션을 더 많이 사용하도록 요청할 것임.

- 자신이 말하는 내용을 화면에 텍스트로 똑같이 표시할 필요는 없다고 생각함.

- 대신, 자신의 말을 강조하는 애니메이션(예시로 현재 보이는 것과 같은)을 더 많이 사용하고 싶음.

- Claude Code에 "제가 말하는 것을 그대로 표시하기보다는 보완하는 애니메이션을 더 많이 사용하고, 모든 애니메이션을 화면 왼쪽으로 중앙 정렬하여 카메라를 오른쪽에 배치할 수 있도록 해주세요."라고 요청할 것임.

- 요청을 보내면 Claude Code가 이전에 했던 작업의 맥락을 가지고 있으므로 편집할 수 있을 것임.

- 이는 한 번의 프롬프트로 끝나는 것이 아니라, 반복적인 작업 과정임을 의미함.

- 원한다면 Remotion 에디터에서 직접 애니메이션의 특정 부분을 축소하거나 삭제하는 등 수정할 수도 있음.

- 마지막으로 렌더 버튼을 누르면 직접 사용할 수 있는 MP4 파일을 얻을 수 있음.

- 에디터가 매우 간단하고 빠르게 작동하여 사용하기 쉬움.

## **개선된 애니메이션 검토 및 추가 피드백**

- Claude Code가 요청한 변경 사항으로 모두 업데이트되었다고 알림.

- 새로운 인스턴스를 재생하면 다음과 같은 변화를 볼 수 있음.

- 애니메이션이 이제 왼쪽에 배치되어 있음.

- 텍스트 대신 그래픽이 더 많이 나타남.

- 다른 그래픽들도 추가되어 있음.

- 하지만 여전히 약간의 텍스트가 필요할 수 있다고 생각함.

- 현재 그래픽에 주로 초점을 맞추고 있으나, 일부 텍스트를 추가하여 강조하는 것이 더 좋을 수 있음.

- 또한, 각 애니메이션이 약 3초 간격으로 나타나는 대신, 자신이 말하는 내용에 맞춰 정확한 타임스탬프를 추가해야 할 것임.

- 아직 약간의 작업이 더 필요하며, 시간을 들여 이 부분을 수정할 예정임.

- 하지만 이 도구가 무엇에 사용되는지, 그리고 얼마나 강력한 애니메이션을 즉시 AI로 만들 수 있는지에 대한 핵심을 이해하길 바람.
