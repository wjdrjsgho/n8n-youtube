## **클로드 4.6 업데이트와 주요 특징**

- 최근 AI 사용자들 사이에서 최추피티(ChatGPT)나 재미나이(Gemini) 대신 클로드(Claude)로의 전환이 활발함.

- 2026년 4.6 버전 업데이트로 클로드의 기능이 대폭 개선됨.

- 주요 개선 사항은 다음과 같음:

- <strong>글쓰기 품질 향상</strong>: 대본, 책 원고, 강의 자료, 제안서 등 글쓰기 작업에 매우 높은 퀄리티를 제공함.

- <strong>스킬 및 코워크 기능 도입</strong>: 업무 방식을 저장하고 PC 파일을 직접 정리해 주는 기능이 추가됨.

- <strong>코딩 실력 세계 1위</strong>: 코딩 실력 평가에서 현재 세계 1위를 기록하여 개발자들 사이에서 '클로드 코드'로 명성이 높음.

- 클로드 코워크 기능은 지난 영상에서 설명되었으며, 이는 클로드가 사용자의 PC 폴더에 직접 접근하여 파일을 읽고, 정리하고, 문서로 만들어 주는 기능임.

- 파일 자동 요약, 영수증/이미지 사진 정리 후 엑셀 변환 등이 가능함.

## **클로드 기본 설정 및 기능 활용법**

- 본격적인 클로드 기능 활용 전, 설정 탭 조정이 필요함.

- <strong>일반 탭 - 맞춤 설정</strong>:

- 사용자의 이름, 직업, 주로 하는 일, 클로드에게 원하는 말투 등 개인 정보를 입력하여 매 대화에 자동으로 적용되게 함.

- 클로드에게 자신의 이야기를 간단히 쓰고 개인 맞춤 설정 멘트를 작성해 달라고 요청할 수도 있음.

- <strong>기능 탭 - 메모리 기능</strong>:

- '채팅 검색 및 참조', '채팅 기록에서 기억 생성' 기능을 활성화하면 이전 대화 내용을 아주 잘 기억함.

- 다른 사람과 공유하지 않고 혼자 사용하는 경우 이 기능을 활성화하는 것이 좋음.

- <strong>커넥터 기능</strong>:

- 구글 드라이브, 지메일 등 외부 서비스를 클로드와 연결하는 곳임.

- 구글 계정과 연결 시 '내 구글 드라이브에서 지난달 보고서 찾아줘'와 같은 요청이 가능해짐.

- <strong>클로드 인 크롬 (Claude in Chrome)</strong>:

- 설치 및 활성화 시, 클로드가 직접 사이트에 접속하여 클릭, 입력 등 상호작용을 통해 정보를 가져옴.

## **클로드의 핵심 기능: 글쓰기와 모델 버전**

- 클로드의 핵심 기능 중 하나는 뛰어난 글쓰기 능력임.

- 실제 말하는 것처럼 자연스럽게 글을 작성함.

- 대본 작업, 책 원고, 강의 자료, 제안서 등 글쓰기가 핵심인 작업에 현존 AI 중 클로드가 가장 낫다고 평가함.

- 26년 2월 기준으로 클로드는 <strong>오퍼스(Opus) 4.6</strong>과 <strong>소넷(Sonnet) 4.6</strong> 버전으로 업데이트됨.

- <strong>오퍼스 4.6</strong>:

- 최고 성능 모델로 복잡하고 어려운 작업에 사용됨.

- 계약서 검토, 긴 보고서 분석, 여러 단계를 거쳐야 하는 기획서 작성에 특히 유리함.

- 생각하는 시간이 더 걸리고 토큰 소모량이 많지만, 결과물의 정교함이 매우 높음.

- <strong>소넷 4.6</strong>:

- 일상적인 업무에 주로 사용됨.

- 대본 초안, SNS 문구, 이메일, 간단한 분석 등에 적합함.

- 오퍼스보다 답변 속도가 빠르고 토큰 소모량이 오퍼스의 약 1/5 수준임.

- 성능 차이는 생각보다 크지 않아 기본적인 글쓰기에는 소넷으로도 충분함.

- <strong>확장 사고 기능</strong>: 토큰 소모량이 많은 오퍼스보다는 소넷 모델에서 '확장 사고'를 클릭하는 것이 더 효율적일 때가 많음.

## **프로젝트 기능 활용법**

- 클로드를 제대로 활용하려면 프로젝트 기능을 만들어 두는 것이 좋음.

- 프로젝트는 사용자의 '전용 작업 공간'으로 볼 수 있음.

- 목적별로 여러 프로젝트를 생성할 수 있음 (예: 유튜브 대본 작성 프로젝트).

- 각 프로젝트 내에 <strong>지침(Instructions)</strong>을 설정하여 프로젝트의 목적, 타겟 독자, 문체, 톤 등을 미리 정리해 두면 매번 질문할 때마다 이를 작성할 필요가 없어짐.

- 지침을 잘 만들어 놓으면 역할 지정이나 상황 설명을 반복할 필요 없이 원하는 형태의 글쓰기 대본 등을 얻을 수 있음.

- 클로드에게 '프로젝트에 들어갈 세부 지침을 만들어 줘'라고 질문하여 지침 작성을 도움받을 수도 있음.

## **클로드 스킬 기능 심층 분석 및 활용 예시**

- 클로드 스킬 기능은 AI를 사용자 업무에 맞게 최적화하여 사용하는 핵심 비결임 (무료 버전에서는 사용 불가, 유료 결제 필요함).

- 스킬은 '이 작업을 할 때는 항상 이렇게 해 줘'와 같은 구체적인 작업 방식을 저장해 두는 기능임.

- <strong>프로젝트와 스킬의 차이점</strong>:

- <strong>프로젝트</strong>는 전용 작업 공간 (목적 중심)임.

- <strong>스킬</strong>은 구체적인 작업 방식 (방법 중심)을 저장함.

- <strong>스킬 생성 방법</strong>:

- 왼쪽 아래 로고 클릭 > 설정 > 기능 탭 > '스킬'에서 '추가' 버튼을 눌러 스킬을 만들 수 있음.

- 예시 스킬 중 <strong>스킬 크리에이터</strong>를 활성화하면 클로드와의 대화를 통해 스킬 생성을 도움받을 수 있음.

- '함께 만들어 보자' 대화형으로 스킬을 창작하거나, 지침을 직접 입력하여 만들 수 있음.

- <strong>스킬 활용 예시 1: 뉴스레터 제작</strong>

- 클로드와의 대화를 통해 뉴스레터 제작 스킬 초안을 완성함.

- 이전에 작성했던 뉴스레터를 참고하도록 입력하여 스타일을 적용함.

- 스킬로 저장 후 '클로드 최신 업데이트에 대한 뉴스레터를 만들어 줘'라고 질문만 하면, 아무런 조건 없이 설정된 스킬에 따라 대상, 문체, 구성이 적용된 뉴스레터 초안을 생성함.

- 고객 견적 안내 문자, 제안서 작성 등 다양한 업무별 스킬을 만들 수 있음.

- <strong>스킬 활용 예시 2: 카드 뉴스 제작</strong>

- 설정 > 사용자 지정 > 스킬 탭에서 카드 뉴스 스킬을 새로 만듦 (스킬 이름은 영문으로 작성).

- 지침란에 클로드에게 요청하여 얻은 카드 뉴스 작성 지침을 복사하여 붙여넣음.

- 스킬 생성 후 '2026년 최신 AI를 활용해서 업무 시간 줄이는 비즈니스 팁 카드 뉴스 만들어 줘'라고 질문하면, 스킬을 자동 활용하여 표지, 본문, 마무리까지 카드 뉴스 텍스트를 바로 생성함.

- 생성된 내용은 수정 요청으로 변경할 수 있으며, 캡처하여 인스타그램이나 블로그에 바로 올릴 수 있음.

- 스킬 세팅 후에는 주제만 바꿔서 질문하면 내 업종에 맞는 다양한 카드 뉴스를 즉시 제작할 수 있음.

## **개발자를 위한 클로드 코드**

- 개발자들 사이에서 '클로드 코드'가 가장 핫한 기능 중 하나임.

- 코딩 지식이 전혀 없어도 말로 지시하면 클로드가 직접 코드를 작성함.

- 현재 브라우저에서 설치 없이 바로 쓸 수 있는 맛보기 버전이 있음.

- 진정한 클로드 코드는 <strong>터미널 기반</strong>으로 설계됨.

- 터미널에서 사용 시 내 컴퓨터 파일 전체를 읽고, 여러 파일을 동시에 수정하며, 결과를 바로 실행까지 할 수 있음.

- macOS에 최적화되어 있으며, 훨씬 안정적이고 할 수 있는 작업 범위도 훨씬 넓다고 함 (Windows에서도 사용 가능함).

- 이러한 이유로 많은 개발자가 맥(Mac)을 주로 사용함.

## **클로드의 단점 및 비용**

- <strong>토큰 소모량</strong>: 클로드는 작업을 수행할수록 '토큰'(사용량)을 많이 소모함.

- 월 20달러 프로 버전 사용자도 코워크로 파일 정리하거나 오퍼스 4.6으로 긴 문서 작업을 하다 보면 토큰 한도가 생각보다 빠르게 소진될 수 있음.

- 클로드를 본격적으로 업무에 활용하는 개발자들은 상당수가 월 100달러 또는 200달러짜리 <strong>맥스 플랜</strong>을 사용하며, 심지어 맥스 플랜을 여러 개 사용하는 경우도 많다고 함.

- 이는 클로드의 기능들이 현장에서 매우 유용하다는 것을 반증함.

## **핵심 요약**

- <strong>모델 선택</strong>: <strong>오퍼스 4.6</strong>은 복잡하고 어려운 작업에, <strong>소넷 4.6</strong>은 일상 업무에 상황에 맞춰 골라 사용할 것임.

- <strong>프로젝트 기능</strong>: 나만의 전용 작업 공간을 먼저 만들어 두고 활용할 것임.

- <strong>스킬 기능</strong>: 자주 쓰는 업무 방식을 저장해 두면 매번 같은 설명을 반복할 필요 없이 사용 가능함.

- <strong>클로드 코드</strong>: 코딩을 몰라도 파일 업로드 후 말로 지시하면 되며, <strong>macOS 터미널</strong>에서 최적의 성능을 발휘함.
