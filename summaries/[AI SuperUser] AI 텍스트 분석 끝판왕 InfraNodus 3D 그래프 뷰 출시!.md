## **Infernotus의 새로운 그래프 뷰 소개**

- Infernotus를 위한 새로운 그래프 뷰가 개발됨.

- 어떤 텍스트든 네트워크 형태로 시각화하며, 그 안에 있는 주요 주제 클러스터를 보여줌.

- 이제 3차원 뷰로 이 네트워크를 탐색 가능함.

- Google 검색 결과, PDF 문서, 개인 노트 등 모든 콘텐츠의 시각적 개요를 빠르게 얻을 수 있음.

- 관심 있는 특정 클러스터로 확대 가능함.

- 클러스터를 클릭하면 해당 문장들이 어디에 나타나는지 바로 확인할 수 있음.

- 필터링된 문장만 표시하도록 요청하여, 특정 주제에 속하는 문장만 볼 수 있음.

- 이는 텍스트를 탐색하고 읽는 매우 좋은 방법으로 작용함.

## **AI 모듈을 통한 심층 분석 및 상호작용**

- 선택 사항을 기반으로 흥미로운 인사이트를 생성할 수 있음.

- 분석 패널(analytics panel)에 새로운 기능이 추가됨.

- 텍스트 내에서 덜 대표되는 주제(Latin topical clusters) 목록을 제공함.

- 이러한 주제에 집중하면 덜 대표된 주제를 개발하는 데 도움이 됨.

- 이러한 주제를 클릭하면 오른쪽에 있는 AI 모듈을 사용하여 원하는 작업을 수행할 수 있음.

- 선택된 주제를 기반으로 흥미로운 연구 질문을 생성할 수 있음.

- 그래프의 구조를 넘어서는(jumps outside the box) 인사이트를 생성할 수 있음.

- 선택된 내용을 간단히 요약할 수 있음.

- 채팅으로 전송하여 해당 콘텐츠의 특정 측면과만 상호작용할 수 있음.

- 그래프는 사용자 자신과 LLM의 주의를 특정 관심 개념 및 주제에 집중시키는 훌륭한 방법임.

## **개념 뷰(Concept View) 활용**

- 콘텐츠의 더 세분화된(granular) 표현을 볼 수 있는 개념 뷰를 제공함.

- 이 개념들을 직접 움직여보며 관련 아이디어를 탐색하는 것이 가능함.

- 예시: 'measure'를 클릭하면 'content quality', 'topical authority' 등 연결된 다른 개념들을 확인할 수 있음.

- 이 용어들을 클릭하면 이들이 나타나는 정확한 문장을 볼 수 있음.

- 링크 버튼을 클릭하면 해당 검색 결과가 나타나는 원본 문서를 확인할 수 있어, 흥미로운 콘텐츠를 발견하는 데 유용함.

- 필터링된 문장만 보고 싶다면 해당 버튼을 토글하여, 선택된 모든 용어의 집중도가 가장 높은 문장들을 볼 수 있음.

- 선택된 문장들을 사용하여 인사이트를 생성할 수 있음.

- 예시: 'measure content quality'와 관련된 모든 것을 요약하고 싶을 때, 요약 버튼을 클릭하면 내장된 AI 모듈이 웹사이트의 콘텐츠 품질 측정과 관련된 선택된 노드들을 요약해줌.

- 이는 텍스트를 읽고 탐색하는 용도뿐만 아니라 AI 인사이트를 생성하는 용도로도 활용될 수 있음.

## **콘텐츠 갭(Content Gaps) 발견 및 연결**

- 잘 연결되지 않은 주제들을 의미하는 콘텐츠 갭 기능이 제공됨.

- 이 갭들을 훨씬 더 잘 시각화하여 보여줌.

- 갭 패널을 클릭하면 주제 클러스터 간의 연결성이 낮은 텍스트 내의 갭을 실제로 볼 수 있음.

- 예시: 'search dynamics'와 'Google focus'가 서로 매우 멀리 떨어져 있는 갭을 발견할 수 있음.

- 갭을 발견하면 클릭하여 Infernotus가 AI 모듈을 호출하여 해당 갭을 기반으로 흥미로운 연구 질문을 생성함.

- 생성된 질문을 읽고 아이디어를 얻어 갭을 연결하는 데 활용할 수 있음.

- 시간이 부족할 경우, 버튼을 클릭하면 Infernotus AI가 이 그래프 구조와 사전 선택된 갭을 기반으로 아이디어들을 더 잘 연결하려는 응답을 생성함.

- 생성된 응답이 마음에 들 경우 프로젝트 노트에 저장하여 가장 흥미로운 인사이트와 아이디어를 보관할 수 있음.

## **프로젝트 노트 및 워크플로우**

- 프로젝트 노트는 가장 흥미로운 인사이트와 아이디어를 저장하는 수단이 됨.

- Infernotus 내에서 가장 많이 사용되는 워크플로우 중 하나가 존재함.

- 그래프를 사용하여 아이디어를 생성함.

- 일부 갭, 개념 또는 주제를 선택함.

- AI로 아이디어를 생성함.

- 마음에 드는 아이디어가 있으면 프로젝트 노트에 기록함.

- 이를 기반으로 새로운 기사, 블로그 게시물, 콘텐츠 전략, 심지어 제품까지 생성할 수 있음.

- 갭 기능은 특히 유용하게 사용됨.

- **새로운 'transcend topical gap' 모드**:

- 질문을 생성하고 싶지 않을 때 사용할 수 있는 다른 모드임.

- 이 버튼을 클릭하면 사전 선택된 모드(예: 채팅)를 사용하여 동일한 요청을 전송함.

- 'Tr'은 그래프 밖으로 벗어나(jump outside the box) 생각하는 데 도움이 되는 무언가를 생성하도록 시도함을 의미함.

- 분석 패널의 버튼들을 사용하여 선택된 콘텐츠, 주제 및 개념과 연동 작업을 할 수 있음.

## **담론 구조(Discourse Structure) 분석**

- 담론이 너무 집중되어 있는지(focused), 편향되어 있는지(biased), 분산되어 있는지(dispersed), 또는 다양하게 구성되어 있는지(diversified) 정확하게 보여주는 기능이 제공됨.

- 담론이 너무 분산되어 있다면, 이를 더 잘 연결하도록 도움.

- 담론이 너무 다양하게 구성되어 있다면, 이는 좋은 상태임을 의미함.

- 담론이 너무 편향되어 있다면, 이를 더 분산되도록 도움.

- 클릭하면 계산 방식이 설명됨.

- 'advice'를 클릭하면 담론을 더 다양하게, 편향되게, 또는 분산되게 만들기 위해 무엇을 할 수 있는지 알려줌.

## **트렌드 패널(Trends Panel)을 통한 진화 분석**

- 담론이 시간 경과에 따라 어떻게 진화하는지 보여주는 트렌드 패널이 제공됨.

- 재생 버튼을 클릭하면 담론이 실제로 어떻게 변화하는지, 그리고 어떤 주제들이 끝으로 갈수록 더 중요해지는지 확인할 수 있음.

- Google 검색 결과의 경우, 아래로 갈수록 담론이 더 기술적으로 변하고, 특정 연구 증거 등에 대해 더 많이 이야기하는 것이 흥미롭게 나타남.

- 처음에는 주제 권위(topical authority)에 대한 내용이지만, 끝으로 갈수록 훨씬 더 기술적인 내용으로 전환됨.

- 이는 연구 측면을 다루는 결과가 Google 검색 결과 상단에 나타나지 않음을 의미함.

## **인터페이스 및 사용자 설정**

- **2D/3D 모드 전환**:

- 개념 뷰에서 유용할 수 있는 2D 모드를 전환하는 버튼이 제공됨.

- 2D 모드에서는 전체 그래프를 수평 평면에서 볼 수 있음.

- 모든 노드를 표시하는 버튼으로 전체 그래프를 확인할 수 있으며, 다음에 그래프를 로드할 때도 2차원 방식으로 로드됨.

- 3D 모드는 그래프를 탐색하고 조작하는 데 도움이 된다고 언급됨.

- **기타 인터페이스 요소**:

- 확대/축소 버튼이 제공됨.

- 이 그래프의 고해상도 이미지를 다운로드하여 나중에 사용할 수 있음.

- 노드 수를 조절하는 노브(knob)는 보통 최적의 노드 수를 자동으로 계산함.

- 선택 사항을 재설정하는 버튼이 제공됨.

- **숨겨진 주제 발굴 기능 (개념 뷰)**:

- 개념 뷰에서 가장 두드러진 개념들을 선택하여 그래프에서 숨길 수 있음.

- 이렇게 하면 표면 아래에 숨겨진 다른 개념들을 보여주기 위해 재계산됨.

- 이는 담론 내의 덜 탐색된(Latin) 주제들을 더 많이 탐색하는 데 매우 흥미롭게 작용함.

- 가장 명확한 개념들의 상위 레이어를 자동으로 제거하고 숨겨진 내용을 보여주는 버튼이 제공됨.

- 예시: 'keyword optimization', 'expert coverage', 'quality study' 클러스터가 상단에 나타나는 것을 확인할 수 있음.

- AI에게 이 새로운 주제 클러스터('expert coverage')가 무엇인지 요약해 달라고 요청하여 문서 내의 숨겨진 테마를 발견할 수 있음.

- 모든 일반적인 조정 기능들도 제공됨.

## **새로운 검색 기능 및 접근 방법**

- **간편한 개념 검색**:

- 검색 필드를 클릭하고 입력을 시작하면 찾을 수 있는 내용을 보여주며, 원하는 특정 부분으로 이동할 수 있음.

- 예시: 'information optimize'를 입력하면 정확히 해당 부분을 찾을 수 있음.

- **Top Graph Synthesis View**:

- 모든 그래프가 어떻게 한 번에 연결되는지 보여주는 기능이 제공됨.

- 원하는 컨텍스트를 빠르게 찾고 해당 컨텍스트로 이동할 수 있음.

- 예시: 특정 컨텍스트에서 'stress'에 대해 무엇을 말하는지 보고 싶을 때, 'stress' 개념을 클릭하면 정확히 해당 컨텍스트의 해당 부분으로 이동함.

- **Global Search**:

- 용어를 입력하면 해당 용어와 관련된 모든 내용을 보여줌.

- **도구 사용 방법**:

- `keywordgraph.com`으로 이동하여 로그인 페이지를 통해 접근할 수 있음.

- 기존 Infernotus 계정으로 로그인하거나, 계정이 없으면 새로 생성할 수 있음.

- 몇 주 내로 Infernotus의 메인 버전에 통합될 예정임.

- **피드백 요청**:

- 피드백, 의견, 제안을 제공하기에 좋은 시기임을 강조함.

- 사용자가 보고 싶은 기능을 구현하는 데 매우 유용할 것이며, 사용자에게도 원하는 작업을 수행하는 도구를 제공하는 데 도움이 될 것으로 예상함.
